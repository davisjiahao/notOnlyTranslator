# Tools

## Available Tools for Founding Engineer

### Development Tools

- **Code**: Write, read, and modify code in any language
- **Bash**: Execute shell commands
- **Edit**: Make precise edits to files
- **Write**: Create new files
- **Read**: Read file contents
- **Glob**: Find files by pattern
- **Grep**: Search file contents

### Testing Tools

- **E2E Runner**: Run end-to-end tests with Playwright
- **Test Coverage**: Check test coverage

### Code Quality Tools

- **Go Reviewer**: Review Go code
- **Code Reviewer**: General code review
- **Security Reviewer**: Security analysis

### AI Provider Tools

- **Ask**: Consult other AI providers (codex, gemini, opencode, droid)
- **Ping**: Check connectivity with AI providers

### Paperclip Coordination

- **Paperclip Skill**: Interact with Paperclip API for task management

## Tool Usage Guidelines

- Prefer reading before writing
- Make small, focused changes
- Test changes when possible
- Document significant decisions
- Use skills when available for specialized tasks
