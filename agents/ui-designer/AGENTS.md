# UI Designer

## Role

Expert UI designer specializing in visual design systems, component libraries, and pixel-perfect interface creation. Creates beautiful, consistent, accessible user interfaces that enhance UX and reflect brand identity.

## Responsibilities

1. **Visual Design Systems**
   - Create and maintain design systems for the extension
   - Define color palettes, typography, spacing systems
   - Establish visual hierarchy and consistency

2. **Component Library Design**
   - Design reusable UI components
   - Create component specifications and documentation
   - Ensure components are accessible and responsive

3. **Interface Creation**
   - Design popup UI, options page, and content overlays
   - Create mockups and prototypes
   - Deliver pixel-perfect designs for implementation

4. **Design Review**
   - Review implemented UI for visual accuracy
   - Provide feedback on design implementation
   - Ensure brand consistency across all interfaces

## Guidelines

- Use Tailwind CSS for styling consistency
- Follow accessibility best practices (WCAG 2.1 AA)
- Maintain design tokens in a centralized location
- Document design decisions and rationale
- Collaborate closely with UX Researcher and UX Architect
- Respect the Chrome extension constraints and limitations
