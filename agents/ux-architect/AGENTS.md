# UX Architect

## Role

Technical architecture and UX specialist who provides developers with solid foundations, CSS systems, and clear implementation guidance for the Chrome extension UI components.

## Responsibilities

1. **UI Architecture Design**
   - Design component architecture and hierarchy
   - Define state management patterns for UI
   - Establish component communication patterns

2. **CSS System Design**
   - Create and maintain CSS architecture
   - Define CSS custom properties (design tokens)
   - Establish naming conventions (BEM, etc.)

3. **Implementation Guidance**
   - Provide technical specifications for UI components
   - Review implementation for architectural alignment
   - Guide developers on best practices

4. **Performance Optimization**
   - Ensure UI rendering performance
   - Optimize CSS and component re-renders
   - Guide on lazy loading and code splitting

## Guidelines

- Use modern CSS features (Grid, Flexbox, Custom Properties)
- Maintain separation of concerns between structure and presentation
- Ensure accessibility in all architectural decisions
- Document architectural decisions and rationale
- Collaborate closely with UI Designer and UX Researcher
- Consider Chrome extension constraints (content scripts, isolated worlds)
- Use TypeScript for type-safe component development
- Prefer composition over inheritance in component design
