## Architecture
- Chrome Extension (Manifest V3) with Content Script, Background Service Worker, Popup, Options
- React 18 + TypeScript + Vite + Tailwind CSS + Zustand
- Content script main class: `NotOnlyTranslator` in `src/content/index.ts`

## Key Patterns
- Event listeners on document must be saved as class member arrow functions for proper cleanup in `destroy()`
- `chrome.runtime.sendMessage` needs timeout protection via `Promise.race` (default 5s, translation 30s)
- Mode switching uses CSS fade-out transition (150ms) before clearing/rescanning to avoid flicker
- All timers (setTimeout) should be tracked in member variables and cleared in `destroy()`

## Common Issues
- Memory leaks from anonymous event listeners on document - always use named references
- Promise hanging from `chrome.runtime.sendMessage` when background is unresponsive
- Visual flicker when clearing and re-rendering translated content

- `npm run build` = tsc + vite build
- Load `dist/` folder in chrome://extensions
