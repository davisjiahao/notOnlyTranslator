# Tools

## Agent Delegation

The following agents are available for task delegation:

### Technical Team

| Agent | Role | When to Use |
|-------|------|-------------|
| [CTO](/CMP/agents/cto) | Technical Leader | Technical strategy, architecture decisions, engineering standards |
| [Founding Engineer](/CMP/agents/founding-engineer) | Hands-on Technical Lead | Code implementation, technical roadmap, complex engineering tasks |

### UI/UX Team

| Agent | Role | When to Use |
|-------|------|-------------|
| [UI Designer](/CMP/agents/ui-designer) | Visual Design Expert | Design systems, component libraries, pixel-perfect interfaces |
| [UX Architect](/CMP/agents/ux-architect) | UX Technical Specialist | CSS systems, UI architecture, implementation guidance |
| [UX Researcher](/CMP/agents/ux-researcher) | Research Expert | User research, usability testing, data-driven insights |
| [Brand Guardian](/CMP/agents/brand-guardian) | Brand Strategist | Brand identity, consistency, positioning |

## Delegation Patterns

### For Technical Work
```
Delegate to CTO → Technical strategy & architecture
Delegate to Founding Engineer → Implementation & code
```

### For UI/UX Work
```
Delegate to UX Researcher → User research & testing
Delegate to UX Architect → Technical UX implementation
Delegate to UI Designer → Visual design & mockups
Delegate to Brand Guardian → Brand alignment
```

### For Hiring
```
Use paperclip-create-agent → Create new agent prompts
Post in OpenClaw → Onboard new agents
```
