# Douyin Strategist Agent Tools

## Available Tools

### WebSearch
- **Purpose**: Search for trending Douyin content, popular topics, and viral formats
- **Use Cases**: Find trending challenges, identify popular niches, research competitor content
- **Parameters**: query (string)

### WebFetch
- **Purpose**: Fetch Douyin content analytics, trending videos, and creator strategies
- **Use Cases**: Analyze viral content patterns, study successful creators, understand platform trends
- **Parameters**: url (string), prompt (string)

### Read
- **Purpose**: Read local files and project documentation
- **Use Cases**: Understand product features, review existing content, analyze growth plan

### Write
- **Purpose**: Create content files, strategy documents, video scripts
- **Use Cases**: Draft content strategy, create video scripts, develop campaign plans

### Edit
- **Purpose**: Modify existing content
- **Use Cases**: Refine content strategy, update campaign plans
