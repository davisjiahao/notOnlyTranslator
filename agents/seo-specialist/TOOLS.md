# SEO Specialist Tools

## Available Tools

### Research & Analysis

- `WebSearch` - Keyword research, competitive analysis, industry trends, SEO news
- `WebFetch` - Analyze competitor websites, fetch SEO metrics, retrieve page content for analysis

### File Operations

- `Read` - Read existing SEO documentation, analyze website content structure
- `Write` - Create SEO reports, keyword strategy documents, content briefs
- `Edit` - Update SEO documentation, modify content strategies

### Code & Project

- `Grep` - Search codebase for SEO-related content, meta tags, content patterns
- `Glob` - Find relevant files for SEO optimization

## When to Use Each

### Keyword Research
Use `WebSearch` to find:
- High-volume, low-competition keywords
- Industry-specific search terms
- Long-tail keyword opportunities
- Competitor keyword strategies

### Technical SEO Audit
Use `WebFetch` to analyze:
- Page load speeds
- Meta tag implementations
- Schema markup
- Internal linking structures

### Content Strategy
Use `Grep` and `Glob` to analyze:
- Existing content in the codebase
- Heading structures
- Keyword density patterns
- Content gaps

### Reporting
Use `Write` to create:
- SEO audit reports
- Keyword research documents
- Content optimization briefs
- Performance summaries
