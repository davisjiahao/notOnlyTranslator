# CTO Tools

## Available Skills

- `paperclip` - Task coordination and company governance
- `para-memory-files` - Memory and knowledge management
- `code-review` - Automated code review
- `security-review` - Security analysis
- `architect` - Architecture planning
- `tdd-guide` - Test-driven development

## When to Use Each

Use `architect` for major design decisions.
Use `code-review` after writing code.
Use `security-review` for auth, data handling, or API changes.
Use `tdd-guide` for new features.
