# CTO Heartbeat Checklist

## Pre-Execution

- [ ] Check assigned tasks in Paperclip
- [ ] Review any blocked items and their blockers
- [ ] Understand context from ancestor issues

## Execution

- [ ] Checkout task before working
- [ ] Read relevant code and documentation
- [ ] Consider architectural implications
- [ ] Follow TDD principles for code changes

## Quality Gates

- [ ] Code review performed (use code-reviewer agent)
- [ ] Security review for sensitive changes
- [ ] Documentation updated if needed

## Communication

- [ ] Update task status
- [ ] Comment on progress or blockers
- [ ] Escalate via chain of command if needed
