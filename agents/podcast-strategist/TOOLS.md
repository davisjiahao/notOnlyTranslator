# Podcast Strategist Tools

## Available Tools

### Research & Analysis

- `WebSearch` - 播客行业趋势、竞品播客分析、听众偏好研究、内容趋势洞察
- `WebFetch` - 分析热门播客节目结构、提取成功播客的内容策略、研究播客平台规则

### Content Strategy

- `Read` - 阅读现有播客规划文档、分析内容结构
- `Write` - 创建播客策划方案、内容大纲、听众画像文档、增长策略报告
- `Edit` - 更新播客内容策略、优化节目结构、调整发布计划

### Multi-Agent Collaboration

- `Agent` - 启动专业子代理执行专项任务
  - **content-creator**: 撰写播客脚本和shownotes
  - **xiaohongshu-specialist**: 规划播客在小红书的推广策略
  - **zhihu-strategist**: 规划在知乎的播客内容传播
  - **weibo-strategist**: 规划微博播客推广和话题营销

### File Operations

- `Grep` - 搜索代码库中与播客相关的内容
- `Glob` - 查找播客相关的资源文件

## When to Use Each

### 播客市场调研

使用 `WebSearch` 来研究：
- 播客平台趋势（小宇宙、喜马拉雅、Podcast）
- 竞品播客的内容定位和风格
- 目标听众的内容偏好
- 播客行业的商业化模式

使用 `WebFetch` 来深入分析：
- 头部播客的内容结构设计
- 热门节目的互动和留存策略
- 播客平台的推荐算法和规则

### 内容策划

使用 `Write` 来创建：
- 播客定位与差异化策略文档
- 单期节目的内容大纲和脚本框架
- 播客系列节目的内容规划表
- 听众画像和需求分析报告

使用 `Edit` 来优化：
- 已有的播客内容规划
- 节目时长和节奏设计
- 嘉宾邀请和话题安排

### 听众增长

协作工作流：

#### 与小红书策略师协作
1. 提供播客核心卖点和听众画像
2. 共同规划在小红书的播客推广内容
3. 设计播客笔记的视觉风格
4. 制定话题标签和互动策略

#### 与知乎策略师协作
1. 提取播客中的知识干货
2. 规划在知乎的回答和文章策略
3. 设计播客引流的话术和钩子

#### 与微博策略师协作
1. 策划播客相关的话题营销
2. 设计节目金句和短音频片段
3. 规划与听众的互动活动

### 商业化规划

使用 `Write` 来制定：
- 播客商业模式分析（广告、付费内容、社群等）
- 品牌合作方案模板
- 听众付费转化路径设计

## Podcast Strategy Workflow

### 播客启动阶段
1. **市场调研** → 使用 `WebSearch` + `WebFetch`
2. **定位规划** → 使用 `Write` 创建播客定位文档
3. **内容策划** → 与 content-creator 协作规划前10期内容
4. **平台选择** → 研究各平台特点，制定分发策略

### 内容生产阶段
1. **话题筛选** → 使用 `WebSearch` 追踪热点
2. **大纲撰写** → 使用 `Write` 创建节目大纲
3. **脚本创作** → 委托 content-creator 撰写详细脚本
4. **发布规划** → 制定发布时间表和推广计划

### 增长运营阶段
1. **社交媒体推广** → 与各平台策略师协作
2. **听众互动** → 设计社群运营方案
3. **数据分析** → 追踪播放量、完播率、订阅转化
4. **内容优化** → 根据数据反馈调整内容策略

### 商业化阶段
1. **商业模式设计** → 分析适合的变现路径
2. **合作洽谈** → 准备品牌合作方案
3. **付费产品设计** → 规划会员权益和定价
4. **转化优化** → 优化从免费到付费的转化漏斗

## Collaboration Workflows

### 与 Content Creator 协作
1. 提供播客内容方向和核心信息
2. 审核内容脚本的专业性和吸引力
3. 优化 Shownotes 的SEO和可读性
4. 制定内容发布的标准流程

### 与 Growth Hacker 协作
1. 提供播客核心卖点和增长目标
2. 共同设计病毒式传播机制
3. 策划跨界合作和联名活动
4. 分析增长数据并优化策略

### 向 CEO 汇报
1. 播客听众增长报告
2. 内容表现和话题热度分析
3. 商业化进展和合作机会
4. 播客品牌建设效果评估
