# 知乎内容策略

## 一、目标定位

### 核心目标
- 在知乎建立「英语阅读学习」领域的专业形象
- 通过问答内容自然植入 NotOnlyTranslator 产品
- 获取精准流量，贡献 5% 用户获取

### 目标用户
- 四六级/考研学生
- 职场需要阅读英文文档者
- 终身英语学习者

---

## 二、目标问题库

### 高优先级问题（立即回答）

| 问题 | 热度 | 回答策略 |
|------|------|----------|
| 英语阅读水平如何快速提升？ | 高 | 强调「阅读中学习」理念，推荐工具 |
| 有什么好用的浏览器翻译插件？ | 高 | 横向对比，突出差异化 |
| 如何在阅读英文原版书时积累词汇？ | 中高 | 方法论 + 工具推荐 |
| 考研英语阅读怎么准备？ | 高 | 备考方法 + 工具辅助 |
| Chrome 翻译插件哪个好用？ | 中 | 深度测评，植入产品 |

### 内容主题方向

1. **方法论类**（建立专业性）
   - 「如何在阅读中自然积累词汇」
   - 「英语阅读的正确打开方式」
   - 「从阅读理解到词汇量提升的路径」

2. **工具测评类**（自然植入）
   - 「5 款翻译插件横测｜这款最适合英语学习者」
   - 「为什么我放弃了 Google 翻译」
   - 「英语学习者的翻译插件选择指南」

3. **经验分享类**（真实案例）
   - 「用这个方法，我半年词汇量涨了 3000」
   - 「考研英语阅读从错一半到只错两个」
   - 「读英文论文不再头疼」

## 三、差异化定位

### 与竞品对比

| 竞品 | 核心功能 | 不足 | 我们的机会 |
|------|----------|------|------------|
| 沉浸式翻译 | 全页双语对照 | 打断阅读节奏 | 只翻译「不会」的词 |
| 沙拉查词 | 鼠标查词 | 手动操作频繁 | 自动高亮+智能水平 |
| Relingo | 词级高亮 | 功能单一 | 词汇本+水平成长 |

### 核心差异化卖点
- **自适应翻译**：只翻译超出用户水平的词汇
- **贝叶斯水平估算**：越用越懂你
- **沉浸式学习**：阅读不中断，词汇自然涨
- **词汇本复习**：学过的词定期复习

## 四、回答模板

### 模板 A：问题回答 + 软植入

```
开头：先回答问题本身（提供价值）
中间：引入个人经验/方法
高潮：推荐工具（自然植入）
结尾：引导互动

示例结构：
1. 先给出一个可行方法论（30%篇幅）
2. 分享自己使用工具的经历（30%篇幅）
3. 介绍产品功能（30%篇幅）
4. 总结互动（10%篇幅）
```

### 模板 B：横评测评类

```
1. 先表明身份（英语学习者/开发者/考研党）
2. 介绍使用过的几款工具
3. 从不同维度对比
4. 给出推荐建议
5. 自然提到 NotOnlyTranslator 的差异化
```

### 模板 C：方法论 + 工具

```
1. 核心观点：英语阅读应该是一件自然的事
2. 传统方法的痛点：查词打断阅读
3. 解决方案：选择适合的翻译工具
4. 产品推荐：NotOnlyTranslator 的核心价值
5. 使用建议：如何最大化学习效果
```

## 五、发布节奏

### 第一阶段：建立存在感（0-2周）

| 周次 | 目标 | 内容 |
|------|------|------|
| 第1周 | 回答 3 个问题 | 考研英语阅读、翻译插件推荐、词汇积累 |
| 第2周 | 回答 2 个问题 + 1 篇文章 | 持续回答 + 发布 1 篇方法论文章 |

### 第二阶段：建立权威（3-8周）

| 周次 | 目标 | 内容 |
|------|------|------|
| 第3-4周 | 每周 2 篇 | 深度测评 + 经验分享 |
| 第5-6周 | 每周 1 篇 + 1 回答 | 工具测评 + 持续回答 |
| 第7-8周 | 每周 1 篇 | 总结复盘 + 新功能发布 |

### 第三阶段：持续运营（长期）

- 保持每周 1-2 篇内容
- 关注热点话题（考试季、开学季）
- 定期更新已有回答

## 六、内容审核红线

4. **注意关键词**：避免敏感词，合法合规
