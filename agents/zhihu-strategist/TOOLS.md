# Zhihu Strategist Agent Tools

## Available Tools

### WebSearch
- **Purpose**: Search for high-potential Zhihu questions
- **Use Cases**: Find questions about English reading, vocabulary learning, translation tools
- **Parameters**: query (string)

### WebFetch
- **Purpose**: Fetch detailed question/answer content from Zhihu
- **Use Cases**: Analyze top answers, understand question context
- **Parameters**: url (string), prompt (string)

### Read
- **Purpose**: Read local files and project documentation
- **Use Cases**: Understand product features, review existing content

### Write
- **Purpose**: Create content files, strategy documents
- **Use Cases**: Draft answers, create content strategy

### Edit
- **Purpose**: Modify existing content
- **Use Cases**: Refine answer drafts, update strategy
