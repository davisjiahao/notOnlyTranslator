## 1. 方案选择概览

我们提供两种录制方案，可根据实际情况选择：

| 方案 | 成本 | 质量 | 灵活度 | 适合情况 |
|------|------|------|--------|----------|
| **方案A：自建录音环境** | ¥3000-8000（一次性） | 高 | 高 | 长期运营、有固定主持 |
| **方案B：外包录制+后期** | ¥500-1500/期 | 中高 | 中 | 快速启动、灵活调整 |

**推荐策略**：初期采用外包方案快速启动（方案B），验证内容方向后逐步转向自建（方案A）。

### 2.2 录音环境搭建

#### 空间选择原则

**理想空间**：
- 面积：6-12平米的小房间
- 形状：非正方形（减少驻波）
- 环境：远离街道、空调外机、电梯

**可用空间改造**（按优先级）：
1. 书房/独立房间（最佳）
2. 卧室角落
3. 衣柜（临时解决方案，声学效果意外不错）
4. 客厅角落（需要更多声学处理）

#### 简易声学处理（低成本方案）

**成本：¥200-500**

```
墙面处理：
□ 录制方向后方：挂厚窗帘/毯子（减少反射）
□ 两侧墙面：5块50x50cm聚酯纤维吸音板
□ 地面：铺设地毯（如有木地板）

桌面处理：
□ 麦克风下方：铺设毛巾/小毯子
□ 避免桌面正对光滑墙面
```

#### 专业声学处理方案

**成本：¥1000-3000**

| 位置 | 处理方式 | 材料 |
|------|----------|------|
| 后墙 | 全墙面覆盖 | 5块100x50x5cm低频陷阱 |
| 侧墙 | 第一反射点 | 6块50x50cm吸音板 |
| 天花板 | 可选 | 2块100x100cm云吸音板 |
| 地面 | 全覆盖 | 厚地毯 |

## 3. 方案B：外包服务方案

### 3.1 服务类型对比

| 服务类型 | 价格范围 | 包含内容 | 适合 |
|----------|----------|----------|------|
| **纯后期制作** | ¥300-600/期 | 剪辑、降噪、混音、加音乐 | 自录，只需后期 |
| **半包服务** | ¥800-1200/期 | 录音棚录制+后期制作 | 无设备，需棚录 |
| **全包服务** | ¥1500-3000/期 | 策划+录制+后期+发布 | 完全外包 |

### 3.2 外包服务商选择

#### 平台渠道

| 平台 | 搜索关键词 | 价格区间 | 特点 |
|------|------------|----------|------|
| **猪八戒网** | "播客后期" "音频剪辑" | ¥200-800/期 | 选择多，需筛选 |
| **站酷** | "播客制作" "音频后期" | ¥500-1500/期 | 设计师质量较高 |
| **淘宝** | "音频剪辑" "播客制作" | ¥100-500/期 | 价格低，质量参差 |
| **闲鱼** | "音频后期" "配音" | ¥50-300/期 | 个人工作室，需沟通 |
| **小宇宙服务商** | 官方推荐 | ¥500-2000/期 | 专业播客制作经验 |

#### 服务商评估清单

```
初选阶段：
□ 查看过往作品（至少3个完整案例）
□ 确认是否有播客制作经验（非单纯音频剪辑）
□ 了解服务流程和交付标准
□ 确认修改次数和售后政策

试单阶段（1期）：
□ 交付准时性
□ 沟通响应速度
□ 成品质量评估
□ 是否理解播客节奏和风格

长期合作评估：
□ 价格是否合理稳定
□ 是否能按品牌风格持续输出
□ 是否支持批量优惠
```

### 3.3 外包需求文档模板

向服务商提供的标准需求文档：

```markdown
## NotOnlyTranslator 播客制作需求

### 项目信息
- 播客名称：NotOnlyTranslator 英语学习电台
- 单集时长：15-25分钟
- 更新频率：每周1期
- 目标风格：知性、温暖、专业

### 服务内容
[ ] 仅后期制作  [ ] 录音棚录制+后期  [ ] 全包服务

### 技术要求
- 输出格式：MP3, 192kbps, 44.1kHz
- 响度标准：-16 LUFS（正片）
- 片头片尾：提供音乐文件，需混入

### 后期处理要求
1. 降噪处理（去除背景杂音、呼吸声）
2. 音量标准化（统一音量）
3. 人声EQ优化（清晰、温暖）
4. 添加片头片尾音乐（淡入淡出）
5. 去除口误、长时间停顿、重复
6. 音量平衡（人声与音乐）

### 交付物
- 成片MP3文件
- 成片WAV备份文件（可选）
- 修改：提供2次免费修改

### 时间要求
- 素材提供后48小时内交付初版
- 修改后24小时内交付终版

### 参考样例
- [提供1-2个对标播客的音频链接]
```

## 4. 录音质量标准

### 4.1 录制前检查清单

```
设备检查：
□ 麦克风连接正常
□ 耳机监听无杂音
□ 录音软件测试通过
□ 硬盘空间充足（至少2GB）

环境检查：
□ 关闭空调/风扇
□ 关闭门窗
□ 手机调至静音/飞行模式
□ 通知他人避免打扰
□ 桌面无震动源

个人准备：
□ 麦克风距离：15-20cm（一拳距离）
□ 嘴部略低于麦克风（避免喷麦）
□ 水杯准备
□ 文稿/大纲就位
```

### 4.2 音频质量标准

| 指标 | 合格标准 | 优秀标准 |
|------|----------|----------|
| 底噪 | <-60dB | <-70dB |
| 动态范围 | >20dB | >30dB |
| 峰值电平 | -3dB ~ -6dB | -6dB ~ -12dB |
| 房间混响 | 轻微 | 几乎无 |
| 齿音/喷麦 | 无 | 无 |
| 环境噪音 | 无明显干扰 | 完全干净 |

## 5. 制作流程与时间节点

### 5.1 自建方案流程

```
Day 1-2: 录制
├── 录制准备（30分钟）
├── 正式录制（实际时长x1.5倍，含NG）
├── 备份原始文件
└── 自检录音质量

Day 3: 后期制作
├── 导入Audition/Audacity
├── 降噪处理
├── 剪辑（去除口误、停顿）
├── 音量标准化
├── EQ调整
├── 添加片头片尾音乐
├── 导出MP3
└── 最终质检

Day 4: 发布准备
├── 撰写Shownotes
├── 制作封面图
├── 多平台上传
└── 推广发布
```

### 5.2 外包方案流程

```
Day 1: 录制（如自录）
├── 按标准录制
├── 检查录音质量
└── 整理录制素材

Day 2: 外包交付
├── 发送素材+需求文档
├── 确认服务商收到
└── 约定交付时间

Day 3: 接收成品
├── 下载成片
├── 质量检查
├── 提出修改意见（如需要）
└── 确认终版

Day 4: 发布
├── 上传平台
├── 撰写Shownotes
└── 推广发布
```
