## 一、前期准备阶段

### 1.1 脚本确认

- [ ] 视频主题明确（一句话概括）
- [ ] 目标时长确定（短视频2-3分钟/长视频8-12分钟）
- [ ] Hook（前3秒）已设计并验证吸引力
- [ ] 核心卖点不超过3个
- [ ] CTA（行动号召）明确
- [ ] 关键时间节点已标注

### 1.2 素材准备

- [ ] 录屏素材清单已列出
- [ ] 产品界面截图已准备（高清）
- [ ] 真人出镜片段已安排（如需）
- [ ] BGM已选定（2-3个备选）
- [ ] 音效素材已收集
- [ ] 品牌Logo和片头片尾素材已就位

### 1.3 技术准备

- [ ] 录屏软件设置完成（OBS/系统录屏）
- [ ] 分辨率设置为1920x1080
- [ ] 帧率设置为60fps
- [ ] 麦克风测试完成，电平正常
- [ ] 存储空间充足（至少20GB可用）

---

## 二、拍摄/录制阶段

### 2.1 录屏检查

- [ ] 桌面背景整洁（无敏感信息）
- [ ] 浏览器书签栏已隐藏
- [ ] 通知已关闭（勿扰模式）
- [ ] 录屏范围正确（全屏或选定区域）
- [ ] 鼠标轨迹清晰可见
- [ ] 操作节奏适中（不拖沓不匆忙）

### 2.2 音频检查

- [ ] 环境噪音已控制（关门、关窗、关空调）
- [ ] 麦克风位置正确（距嘴15-20cm）
- [ ] 录音电平测试（-12dB to -6dB）
- [ ] 无回音/杂音
- [ ] 每段录音后有5秒静音（方便剪辑）

### 2.3 真人出镜（如有）

- [ ] 光线充足（面部明亮无阴影）
- [ ] 背景简洁专业
- [ ] 衣着整洁（避免条纹/复杂图案）
- [ ] 眼神看向镜头
- [ ] 表情自然，语速适中

---

## 三、粗剪阶段

### 3.1 素材整理

- [ ] 所有素材已导入项目
- [ ] 素材命名规范（类型_日期_序号_描述）
- [ ] 素材箱分类完成
- [ ] 废片已标记删除
- [ ] 音频已同步（独立录音）

### 3.2 结构搭建

- [ ] 按脚本顺序排列素材
- [ ] 片头已添加（3秒）
- [ ] 片尾已添加（8-10秒）
- [ ] 章节标记已设置（长视频）
- [ ] 总时长符合预期

### 3.3 初步剪辑

- [ ] 明显废片已删除
- [ ] 气口和停顿保留（精剪时处理）
- [ ] 逻辑连贯性检查
- [ ] 节奏感初步把控

---

## 四、精剪阶段

### 4.1 画面剪辑

- [ ] 精确剪辑，无多余帧
- [ ] 转场自然（避免花哨效果）
- [ ] B-roll覆盖充分（每个要点都有画面支撑）
- [ ] 关键操作放大展示
- [ ] 关键画面停留3秒以上（方便弹幕互动）

### 4.2 字幕添加

- [ ] 解说字幕完整（白色+黑色描边）
- [ ] 关键词强调字幕（品牌蓝#3B82F6）
- [ ] 标注字幕清晰（圆角矩形底）
- [ ] 无错别字（逐字检查）
- [ ] 时间轴准确，与话音同步
- [ ] 移动端可读性测试

### 4.3 音频处理

- [ ] 人声电平统一（-18dB to -12dB RMS）
- [ ] 背景音乐音量合适（-25dB to -20dB RMS）
- [ ] 背景音乐淡入淡出自然
- [ ] 降噪处理（如需要）
- [ ] 无爆音/削波
- [ ] 整体混音-16 LUFS

### 4.4 视觉效果

- [ ] 色彩校正完成（对比度+5至+10）
- [ ] 锐化适度（+10至+20）
- [ ] 品牌色彩使用一致
- [ ] 动画效果统一（淡入淡出8帧）
- [ ] 水印位置正确（右下角半透明）

## 八、首支视频专项清单

### 「用了10款翻译插件，都不满意...」

**视频类型**：短视频预告（2-3分钟）
**发布时间**：3/17
**核心目标**：制造悬念，引导关注

#### 内容要点

- [ ] Hook（0-15秒）：展示翻译痛点
  - 频繁切屏查词的烦躁
  - 全文翻译导致学习效果差
  - 10款插件对比的失望

- [ ] 悬念设置（15-90秒）：
  - 提及正在开发的解决方案
  - 展示部分界面（打码处理）
  - 强调差异化卖点（只翻译生词）

- [ ] CTA（90-120秒）：
  - 关注账号获取上线通知
  - 评论区征集功能需求
  - 预告正式评测时间

#### 视觉风格

- [ ] 快节奏剪辑（平均3秒/镜头）
- [ ] 大量文字动画强调
- [ ] 悬念感BGM（节奏感强）
- [ ] 封面：神秘感十足，带问号

## 九、常见问题速查

| 问题 | 解决方案 |
|------|----------|
| 录屏文件太大 | 使用HandBrake压缩，保持画质 |
| 人声不清晰 | 检查麦克风位置，添加EQ提升2-4kHz |
| 字幕识别不准 | 使用剪映专业版自动识别后校对 |
| 导出时间太长 | 关闭"以最大深度渲染"，使用硬件加速 |
| 上传后画质差 | 检查码率设置，B站二压后正常 |
| 音频不同步 | 检查帧率一致性，重新同步 |
