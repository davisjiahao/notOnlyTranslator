## 1. 品牌音乐定位

### 1.1 音乐风格方向

**整体风格**：现代、清新、科技感与温度感并存

**核心情绪关键词**：
- 向上 (Uplifting)
- 专注 (Focused)
- 温暖 (Warm)
- 知性 (Intellectual)
- 轻松 (Relaxed)

**音乐元素建议**：
- 主乐器：钢琴、电钢琴、轻电子合成器
- 辅助乐器：轻柔的弦乐垫音、轻打击乐
- 避免：重金属、过于激烈的节奏、悲伤的小调

### 1.2 品牌调性关联

与产品视觉风格保持一致：
- **蓝色 (#3B82F6)**：冷静、专业、可信赖 → 音乐应清晰、不嘈杂
- **白色背景**：简洁、纯净 → 音乐避免过度复杂编排
- **科技感**：AI、智能 → 可融入轻微电子元素
- **学习场景**：专注、沉浸 → 音乐不应分散注意力

## 2. 片头音乐 (Intro)

### 2.1 技术规格

| 参数 | 规格 |
|------|------|
| 时长 | 15-20秒（最长不超过25秒） |
| 采样率 | 44.1kHz |
| 比特率 | 320kbps |
| 格式 | WAV（原始）/ MP3（交付） |
| 音量 | -14 LUFS（略低于正片，预留 headroom） |

### 2.2 结构设计

```
00:00-00:03  品牌音效标识（可选）- 简短的声音Logo
00:03-00:10  主题旋律呈现 - 抓耳但不喧宾夺主
00:10-00:18  渐强过渡 - 为主持人声音做铺垫
00:18-00:20  干净收束 - 留白给主持人开场
```

### 2.3 音乐描述

**开场感觉**：
- 如同清晨第一缕阳光，清新而充满希望
- 暗示"新的一天，新的学习内容"

**具体音乐特征**：
- 速度：中速偏快 (100-120 BPM)，传达积极向上的能量
- 调性：大调（如 C大调、G大调），明亮正向
- 节奏：4/4拍，稳定但不过于强烈
- 动态：从轻柔到逐渐增强，为主持人入场做铺垫

**参考风格**：
- 类似 Headspace 冥想App的开场音乐
- 类似《纽约时报》The Daily播客的开场质感
- 类似轻电子乐（Chill-out / Lo-fi Hip-hop）的氛围

## 3. 片尾音乐 (Outro)

### 3.1 技术规格

| 参数 | 规格 |
|------|------|
| 时长 | 30-45秒（可循环或淡出） |
| 采样率 | 44.1kHz |
| 比特率 | 320kbps |
| 格式 | WAV（原始）/ MP3（交付） |
| 音量 | -16 LUFS（低于正片，柔和收尾） |

### 3.2 结构设计

```
00:00-00:10  承接正片结尾 - 延续片头主题旋律，但更柔和
00:10-00:25  舒缓发展 - 为主持人结束语留出空间
00:25-00:40  渐弱淡出 - 自然结束，余韵悠长
```

### 3.3 音乐描述

**收尾感觉**：
- 如同日落时分，温暖而令人回味
- 暗示"今天的学习暂告一段落，期待下次相见"

**具体音乐特征**：
- 速度：中速 (80-100 BPM)，比片头更放松
- 调性：与片头呼应，可同一调性或关系调
- 动态：从高到低，逐渐淡出，不突然中断

**与片头的呼应**：
- 使用相同的主题旋律，但以更舒缓的节奏呈现
- 形成"首尾呼应"的完整听感
- 增强品牌识别度

## 4. 品牌音效标识 (Sonic Logo)

### 4.1 定义

一个3-5秒的简短声音标识，可在以下场景使用：
- 片头音乐前奏
- 广告/赞助商口播转场
- 社交媒體短音频片段

### 4.2 设计方向

**概念**："理解"的瞬间 —— 从模糊到清晰的声音隐喻

**实现方式建议**：
- 版本A：从轻微模糊/混响的音色逐渐变得清晰
- 版本B：简单的三个音符，代表"Not-Only-Translator"
- 版本C：类似翻书页的微妙音效 + 轻柔音调

## 5. 制作要求

### 5.1 音乐制作方选择

**方案A：正版音乐库授权**
- 推荐平台：Artlist、Epidemic Sound、AudioJungle
- 预算：$50-200/首（买断授权）
- 优点：版权清晰，选择丰富
- 注意：确保获得播客/广播授权

**方案B：定制作曲**
- 推荐渠道：猪八戒、站酷音乐人、豆瓣音乐人
- 预算：¥1000-5000
- 优点：独一无二，完美契合品牌
- 周期：1-2周

**方案C：AI音乐生成**
- 工具：Suno AI、AIVA、Amper Music
- 预算：免费 - $20/月
- 优点：快速、低成本
- 风险：版权状况尚不明确

### 5.2 交付文件清单

交付时需包含以下文件：

```
podcast-music-package/
├── intro/
│   ├── NOT_intro_full.wav          # 完整版（20秒）
│   ├── NOT_intro_short.wav         # 短版（10秒）
│   └── NOT_intro.mp3               # MP3压缩版
├── outro/
│   ├── NOT_outro_full.wav          # 完整版（40秒）
│   ├── NOT_outro_fade.wav          # 预淡出版（30秒）
│   └── NOT_outro.mp3               # MP3压缩版
├── sonic-logo/
│   ├── NOT_logo_v1.wav             # 版本A
│   ├── NOT_logo_v2.wav             # 版本B
│   └── NOT_logo_v3.wav             # 版本C
└── stems/                          # 分轨文件（可选，用于后期微调）
    ├── NOT_intro_stems.zip
    └── NOT_outro_stems.zip
```

### 5.3 版权与授权要求

**必须获得的授权**：
- [ ] 播客使用授权（Podcast License）
- [ ] 商业使用授权（Commercial Use）
- [ ] 全球分发授权（Worldwide Distribution）
- [ ] 永久使用期限（Perpetual）
- [ ] 可编辑/混音授权（Editable/Remixable）

**版权文件保存**：
- 所有授权证书原件保存于 `/legal/music-licenses/`
- 在播客 Shownotes 中可注明 "背景音乐版权所有"

## 6. 音乐与口播配合指南

### 6.1 片头口播脚本模板

```
[音乐起，0-3秒]

[3-10秒 品牌音效/音乐主题]

[10-18秒 音乐渐强]

[18秒 主持人入场]
"欢迎收听 NotOnlyTranslator 英语学习电台，
我是你的学习伙伴 XXX。
在这里，我们一起探索更高效的英语学习方法..."

[音乐在主持人说话时淡出至完全消失]
```

### 6.2 片尾口播脚本模板

```
[主持人结束语]
"...感谢收听今天的节目。
如果你觉得有收获，记得订阅并分享给朋友。
我们下期再见！"

[片尾音乐淡入，0-10秒]

[10-25秒 音乐发展]
"NotOnlyTranslator 英语学习电台，
让英语学习更简单。"

[25-40秒 音乐渐弱淡出]
```

### 6.3 音量平衡建议

**播出时的音量关系**：

| 元素 | 目标音量 | 说明 |
|------|----------|------|
| 正片内容 | -16 LUFS | 标准播客音量 |
| 片头音乐 | -14 LUFS | 略高，吸引注意 |
| 片尾音乐 | -18 LUFS | 略低，柔和收尾 |
| 口播 | -16 LUFS | 与正片一致 |

**淡入淡出时长**：
- 片头：音乐先起，主持人声音淡入（0.5秒）
- 片尾：主持人结束后，音乐淡入（1秒），最后淡出（3-5秒）
