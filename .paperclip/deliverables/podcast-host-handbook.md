## 1. 播客定位与调性

### 1.1 品牌调性

**核心调性**：知性、温暖、专业、亲切

| 维度 | 要求 | 避免 |
|------|------|------|
| 语气 | 像朋友聊天，有温度 | 播音腔、过于正式 |
| 内容 | 有干货，有洞察 | 空洞说教、纯推销 |
| 节奏 | 适中偏慢，给思考空间 |  rushed、信息轰炸 |
| 态度 | 真诚分享，平等交流 | 居高临下、专家姿态 |

### 1.2 目标听众画像

- **考研学生**：22-28岁，时间紧张，急需高效方法
- **外企白领**：25-35岁，有实际需求，注重效率
- **英语爱好者**：各年龄段，对学习方法感兴趣

**听众期待**：
- 听得懂的理论
- 用得上的方法
- 真实的故事
- 不硬广的产品

## 2. 主持风格指南

### 2.1 声音表达

**语速控制**：
- 标准语速：每分钟220-240字
- 重要观点：放慢至180-200字/分钟
- 过渡衔接：保持流畅，不停顿过久

**音量与情绪**：
- 开场：音量适中偏高，有热情
- 正文：自然平稳，保持温暖
- 强调：音量+语速配合，突出关键词
- 结尾：温暖亲和，期待感

**停顿运用**：
- 段落之间：2-3秒停顿
- 重要观点后：1-2秒停顿，让信息沉淀
- 提问后：2秒停顿，引发思考
- 情绪转折前：1秒停顿

### 2.2 语言风格

**用词原则**：
- 口语化但不随意
- 专业术语要解释
- 多用「你」「我们」，拉近距离
- 避免：「众所周知」「显然」等居高临下的词

**句子结构**：
- 多用短句，避免长难句
- 每段不超过3个观点
- 复杂概念分步骤讲解

**示例对比**：

| 避免 | 推荐 |
|------|------|
| 「可理解输入理论是克拉申二语习得理论的核心概念」 | 「克拉申提出了一个叫「可理解输入」的理论——简单说，就是...」 |
| 「全文翻译会导致认知努力的缺失」 | 「全文翻译的问题在于，你的大脑「偷懒」了」 |
| 「建议采用渐进式学习方法」 | 「我的建议是，从简单的开始，一步步来」 |

## 3. 录制准备清单

### 3.1 录制前（30分钟）

**设备检查**：
```
□ 麦克风连接正常
□ 耳机监听无杂音
□ 录音软件测试通过
□ 硬盘空间充足
```

**环境准备**：
```
□ 关闭空调、风扇
□ 关闭门窗
□ 手机调至静音/飞行模式
□ 水杯准备
□ 通知他人避免打扰
```

**个人准备**：
```
□ 充分休息，声音状态良好
□ 润喉，保持口腔湿润
□ 试录1分钟，检查音质
□ 调整心态，进入「分享」状态
```

**内容准备**：
```
□ 完整浏览本期大纲
□ 标记重点段落
□ 熟悉时间戳节点
□ 准备1-2个即兴发挥的点
```

### 3.2 录制中

**麦克风位置**：
- 距离：15-20cm（一拳距离）
- 角度：略低于嘴部，避免正对
- 高度：与嘴角平齐或略低

**状态监控**：
- 保持微笑（会影响声音温度）
- 注意坐姿，保持气息顺畅
- 定时喝水（小口，避免声音变化）
- 注意时间节奏

**NG处理**：
- 口误：停顿1秒，重读这一句
- 长停顿：剪辑时去除
- 状态不佳：暂停休息，重新录制

### 3.3 录制后

**立即检查**：
```
□ 完整回放一遍
□ 检查是否有明显杂音、爆音
□ 确认音量一致性
□ 标记需要剪辑的位置
```

**文件管理**：
- 原始文件命名：`NOT_S01E01_raw_YYYYMMDD.wav`
- 立即备份到云盘
- 记录录制时长和完成时间

## 4. 节目结构导航

### 4.1 标准节目流程

**片头音乐段（00:00-00:30）**
- 配合音乐节奏，准备进入状态
- 音乐淡出时，开始口播

**开场白（00:30-01:30）**
```
结构：
1. 问候语（5秒）
2. 本期价值预告（30秒）
3. 内容概览（25秒）

要点：
- 开场钩子要抓人
- 快速点明本期价值
- 建立期待感
```

**轻互动（01:30-02:00）**
```
形式：
- 提问：「你有没有遇到过...」
- 投票：「A还是B？」
- 引用留言：「上周有听友留言说...」

目的：
- 拉近距离
- 引发共鸣
- 鼓励评论
```

**主要内容（02:00-18:00）**
```
结构要求：
- 每5-7分钟一个小节
- 小节间有过渡句
- 穿插互动提问
- 案例故事穿插

节奏控制：
- 理论+案例+建议
- 信息密度递减：密→疏→密
- 每段结束有小结
```

**互动环节（18:00-21:00）**
```
内容：
- 回应上期评论
- 回答听众问题
- 分享听众故事

要点：
- 念出听友昵称
- 真诚回应
- 引导更多互动
```

**产品植入+CTA（21:00-22:00）**
```
原则：
- 自然融入，不突兀
- 从用户角度出发
- 点到为止，不过度推销

结构：
1. 场景引入（10秒）
2. 产品介绍（20秒）
3. 行动号召（10秒）
```

**结尾（22:00-24:00）**
```
结构：
1. 本期总结（30秒）
2. 下期预告（20秒）
3. 订阅引导（15秒）
4. 感谢语（5秒）

要点：
- 总结要有记忆点
- 预告要制造期待
- 订阅引导要具体
```

### 4.2 不同类型节目调整

**学习方法论类**：
- 理论讲解时间可延长
- 增加案例比重
- 结尾增加「行动清单」

**用户故事类**：
- 减少口播，多让嘉宾说
- 提问要有引导性
- 适时总结嘉宾观点
- 控制总时长（18-20分钟）

**场景实战类**：
- 步骤讲解要清晰
- 可加入「屏幕演示」描述
- 提供可下载的资源链接

## 5. 采访技巧（用户故事系列）

### 5.1 采访前准备

**嘉宾沟通**：
- 提前1周确认时间
- 发送采访大纲
- 确认故事关键节点
- 告知录音时长（30-45分钟原始素材）

**问题设计**：
- 开放式问题为主
- 按时间线设计问题
- 准备3-5个追问点
- 预设情绪高点（感人/励志时刻）

### 5.2 采访中技巧

**开场破冰**：
```
- 简单自我介绍
- 说明节目定位
- 感谢嘉宾参与
- 确认录音开始
```

**提问技巧**：
```
1. 引导式提问：
   「能不能描述一下当时的场景？」

2. 追问式提问：
   「你刚才提到了...能具体说说吗？」

3. 情感式提问：
   「那时候你一定很...？」

4. 总结式提问：
   「如果用一句话总结这段经历，你会说什么？」
```

**倾听与回应**：
- 不打断，让嘉宾完整表达
- 适时「嗯」「对」表示在听
- 捕捉关键词，准备追问
- 嘉宾情绪激动时，给停顿空间

**控场要点**：
- 时间过半时，加速推进关键问题
- 偏离主题时，温和引导回主线
- 嘉宾紧张时，多鼓励，放慢节奏

### 5.3 采访后

**素材整理**：
- 标记精彩段落
- 记录时间戳
- 提取可用引语

**后期配合**：
- 与剪辑师沟通剪辑重点
- 确认嘉宾是否需听审
- 获取嘉宾肖像使用授权

## 6. 产品植入指南

### 6.1 植入原则

**黄金法则**：
- 产品植入时长不超过节目总时长5%
- 必须在提供价值之后植入
- 从用户痛点切入，而非产品功能切入
- 真诚推荐，不过分夸大

**禁止行为**：
- 节目一开头就推销产品
- 长时间罗列产品功能
- 贬低竞品抬高自己
- 虚假承诺效果

### 6.2 植入话术模板

**模板1：解决方案式**
```
"刚才说的这些方法，
如果你觉得自己操作起来有难度，
其实可以试试我们开发的 NotOnlyTranslator。

它会自动识别你词汇量之外的词，
只翻译那些真正需要翻译的内容，
让你在阅读中自然地积累词汇。

官网是 notonlytranslator.com，
可以免费下载试用。"
```

**模板2：用户场景式**
```
"说到这，我想起一个用户的故事。
他之前每天花2小时背单词，
效果却一直不好。

后来他开始用我们的插件阅读英文文章，
三个月后词汇量从5000涨到了8000，
而且阅读速度也明显提升。

如果你也想尝试这种方法，
可以去 notonlytranslator.com 下载。"
```

**模板3：轻植入式**（适合内容密度高的节目）
```
"对了，如果你对我们的方法感兴趣，
可以了解一下 NotOnlyTranslator 这个工具，
它的设计理念就是今天讲的「部分翻译」。

链接在 Shownotes 里，
有需要的朋友可以看看。"
```

### 6.3 CTA 话术

**订阅引导**：
- 「点击订阅，每周一更新不见不散」
- 「觉得有帮助的话，记得订阅这档播客」
- 「不想错过下期内容的话，点个订阅吧」

**评论引导**：
- 「在评论区留下你的问题，我们会挑选回答」
- 「如果你也有类似经历，欢迎在评论区分享」
- 「这期内容对你有帮助吗？在评论区告诉我」

**分享引导**：
- 「把这期内容分享给也在学英语的朋友吧」
- 「如果觉得有价值，欢迎转发给需要的人」

## 7. 常见问题与应对

### 7.1 技术问题

**声音沙哑**：
- 录制前多喝温水
- 准备润喉糖
- 状态不佳时改期录制

**环境噪音**：
- 立即停止录制
- 处理噪音源
- 从上一个段落重新开始

**忘词/卡壳**：
- 停顿1秒，看大纲
- 用自己的话重新组织
- 不要追求完美，自然更重要

### 7.2 内容问题

**内容过长**：
- 精简案例细节
- 压缩过渡段落
- 控制在25分钟内

**内容过短**：
- 增加互动环节
- 延展某个观点
- 补充实用建议

**状态不佳**：
- 深呼吸，调整状态
- 暂停休息5分钟
- 改期录制（如必要）

### 7.3 突发情况

**嘉宾临时缺席**：
- 启用备用单人节目内容
- 或改期发布
- 提前在社交媒体说明

**设备故障**：
- 使用手机备用录音
- 改期录制
- 及时通知团队

## 8. 质量自检清单

### 8.1 录制自检

```
□ 开场是否吸引人？
□ 语速是否适中？
□ 是否有明显口误？
□ 停顿是否自然？
□ 情绪是否饱满？
□ 产品植入是否自然？
□ 结尾是否有力？
□ 总时长是否符合要求？
```

### 8.2 后期反馈

收到剪辑后的成品，检查：
```
□ 剪辑是否流畅？
□ 音量是否一致？
□ 音乐混入是否自然？
□ 有无技术问题？
□ 整体听感是否舒适？
```

## 9. 持续改进

### 9.1 数据反馈

**关注指标**：
- 完播率（目标：>50%）
- 评论率（目标：>3%）
- 订阅转化率（目标：>5%）

**听众反馈**：
- 定期阅读评论
- 收集听众建议
- 调整内容方向

### 9.2 自我提升

**定期复盘**：
- 每期录制后自我评估
- 标记需要改进的点
- 下期针对性调整

**学习资源**：
- 收听优秀播客，学习技巧
- 关注播客制作知识
- 与同行交流经验
