## 一、平台技术规格

### 1.1 上传参数

| 参数 | 推荐设置 | 备注 |
|------|----------|------|
| 分辨率 | 1920x1080 (1080P) | 最低要求，推荐4K供二创 |
| 帧率 | 60fps | 知识区30fps也可接受 |
| 编码格式 | H.264 (AVC) | HEVC兼容性差，避免使用 |
| 码率 | 12-20 Mbps | 1080P推荐15Mbps |
| 色彩空间 | Rec.709 | B站不支持广色域显示 |
| 音频码率 | 320kbps AAC | 清晰人声关键 |
| 文件格式 | MP4 (.mp4) | 兼容性最佳 |
| 文件大小 | 单次最大8GB | 超过需分段上传 |

### 1.2 封面规格

| 参数 | 设置 |
|------|------|
| 尺寸 | 1146 x 717 像素 (16:10) |
| 格式 | JPG/PNG |
| 文件大小 | < 5MB |
| 安全区域 | 中心 1000x600（避免被裁切） |
| 文字要求 | 清晰可读，避免过小字体 |

### 1.3 分P视频规范

```
系列内容建议分P结构：
- P1: 正片（8-12分钟）
- P2: 无BGM纯享版（可选，方便用户跟练）
- P3: 素材/工程文件展示（增强专业感）
```

## 二、剪辑工作流

### 2.1 项目文件夹结构

```
Bilibili_Project_YYYYMMDD/
├── 01_Footage/              # 原始素材
│   ├── ScreenRecord/        # 录屏素材
│   ├── Camera/              # 实拍素材（如有）
│   └── Audio/               # 独立音频
├── 02_Assets/               # 资产文件
│   ├── Logo/                # 品牌Logo
│   ├── LowerThirds/         # 字幕条模板
│   ├── Transitions/         # 转场预设
│   └── BGM/                 # 背景音乐
├── 03_Projects/             # 工程文件
│   ├── Premiere/            # PR项目
│   └── AfterEffects/        # AE合成（如有）
├── 04_Exports/              # 输出文件
│   ├── Draft/               # 草稿版本
│   ├── Review/              # 审核版本
│   └── Final/               # 最终版本
└── 05_Thumbnails/           # 封面设计
```

### 2.2 剪辑流程

```
第一阶段：素材整理（30分钟）
├── 导入所有素材
├── 命名规范：类型_日期_序号_描述
│   例：SR_0315_01_插件界面展示.mp4
├── 创建素材箱分类
└── 同步音频（如有独立录音）

第二阶段：粗剪（1小时）
├── 按脚本结构排列素材
├── 标记卡点位置
├── 删除明显废片
└── 保留气口和停顿（精剪时再处理）

第三阶段：精剪（2-3小时）
├── 精确剪辑，控制节奏
├── 添加B-roll覆盖
├── 插入字幕和标注
├── 添加转场和动效
└── 调整音频电平

第四阶段：包装（1小时）
├── 设计/导入封面
├── 添加片头片尾
├── 统一字幕样式
├── 色彩校正
└── 最终输出
```

## 三、色彩分级规范

### 3.1 录屏素材调色

**屏幕录制常见问题：**
- 色偏（尤其是夜间模式）
- 对比度过高/过低
- 文字边缘锯齿

**标准调色参数（Premiere Lumetri）：**

```
基础校正：
├── 输入LUT: None（屏幕素材不需要）
├── 白平衡: 保持原样（除非明显偏色）
├── 色调:
│   ├── 曝光: 0（屏幕录制通常无需调整）
│   ├── 对比度: +5 至 +10（增强文字清晰度）
│   ├── 高光: -10（防止过曝）
│   ├── 阴影: +5（提亮暗部细节）
│   ├── 白色: 0
│   └── 黑色: +5 至 +10（避免纯黑压文字）
├── 饱和度: -5 至 -10（屏幕色彩通常过艳）
└── 锐化: +10 至 +20（增强文字边缘）
```

### 3.2 品牌色彩规范

**主色调：**
```
品牌蓝:   #3B82F6 (RGB: 59, 130, 246)
辅助绿:   #10B981 (RGB: 16, 185, 129)
背景浅色: #F3F4F6 (RGB: 243, 244, 246)
背景深色: #1F2937 (RGB: 31, 41, 55)
文字深色: #111827 (RGB: 17, 24, 39)
文字浅色: #F9FAFB (RGB: 249, 250, 251)
强调橙:   #F59E0B (RGB: 245, 158, 11) - 用于警告/重要提示
```

**色彩使用规则：**
- 录屏画面保持原色，不调色
- UI元素（字幕条、标注）使用品牌色
- 对比度确保文字可读性（WCAG AA标准）

## 四、音频工程规范

### 4.1 音轨结构

```
Track 1: 主音轨（旁白/解说）
Track 2: 背景音乐（BGM）
Track 3: 音效层（SFX）
Track 4: 系统音效（录屏中的操作声）
Track 5: 备用/音乐过渡
```

### 4.2 音频电平标准

| 音频类型 | 峰值电平 | RMS电平 | 备注 |
|----------|----------|---------|------|
| 人声旁白 | -3dB | -18dB to -12dB | 清晰为主 |
| 背景音乐 | -20dB | -25dB to -20dB | 不压人声 |
| 音效提示 | -10dB | - | 短暂出现 |
| 整体混音 | -1dB | -16dB LUFS | 符合B站规范 |

### 4.3 音频处理流程

```
人声处理链：
1. EQ (均衡器)
   ├── 高通滤波: 80Hz cutoff（去除低频噪声）
   ├── 中频提升: 2-4kHz +2dB（增加清晰度）
   └── 高频衰减: >8kHz -1dB（减少齿音）

2. 压缩器
   ├── 阈值: -18dB
   ├── 比率: 3:1
   ├── 启动时间: 5ms
   ├── 释放时间: 50ms
   └── 增益补偿: +3dB

3. 降噪（如需要）
   └── 轻度降噪，不影响人声质感

4. 限制器
   └── 阈值: -3dB（防止爆音）
```

### 4.4 BGM选择指南

**Bilibili知识区常用风格：**
- **轻快向：** Lofi Hip Hop, Chillhop
- **科技向：** Electronic Ambient, Synthwave
- **学习向：** Piano Solo, Acoustic Guitar
- **节奏向：** Upbeat Corporate, Motivational

**推荐曲库来源：**
- YouTube Audio Library（免费）
- 爱给网（aigei.com）
- 淘声网（tosound.com）
- Epidemic Sound（付费）

**使用原则：**
- 同一视频不超过2首BGM切换
- 音量始终低于人声12dB以上
- 片头15秒和结尾15秒可独立配乐

## 五、字幕与标注规范

### 5.1 字幕样式

**主字幕（解说）：**
```
字体：思源黑体 / 阿里巴巴普惠体
字号：根据画面调整，确保移动端可读
颜色：白色 (#FFFFFF) + 黑色描边 3px
位置：画面下方 1/4 处
动画：淡入淡出 8帧
持续时间：至少2秒，跟随语速
```

**强调字幕（关键词）：**
```
字体：同上，加粗
颜色：品牌蓝 (#3B82F6)
动画：弹入 + 轻微缩放
使用场景：产品名称、核心卖点、数据
```

**标注字幕（画面指引）：**
```
样式：圆角矩形底 + 白色文字
背景：半透明黑色 80% 或品牌色
箭头：使用简洁线条箭头
持续时间：跟随被标注元素存在时长
```

### 5.2 弹幕引导设计

**关键时间点弹幕预设：**
```
00:30 - 片头结束: "正片开始"
功能演示前: "注意看这个细节"
数据展示时: "截图保存"
产品对比时: "种草了"
安装教程时: "已安装"
片尾: "一键三连"
```

**在视频中预留弹幕互动空间：**
- 关键画面停留3秒以上
- 在画面边缘预留弹幕覆盖区域
- 避免重要信息被弹幕遮挡

## 六、片头片尾模板

### 6.1 片头模板（3秒）

**结构：**
```
[0:00-0:00.5] 黑屏 → Logo淡入
[0:00.5-0:02] Logo动画 + 品牌音效
[0:02-0:03] 标题文字滑入
[0:03] 切入主画面
```

**技术参数：**
- Logo动画：缩放+淡入，0.5秒
- 过渡：快速模糊溶解
- 音乐：品牌音效（0.5秒）

### 6.2 片尾模板（8-10秒）

**结构：**
```
[0:00-0:02] 主画面渐隐 → 片尾背景
[0:02-0:04] "感谢观看" + 点赞图标动画
[0:04-0:06] "关注获取更多精彩" + 关注按钮
[0:06-0:08] 其他视频推荐缩略图
[0:08-0:10] Logo + Slogan
```

**固定元素：**
- 品牌Logo
- B站关注二维码/账号名
- 一键三连引导动画

## 七、导出设置

### 7.1 Premiere Pro 导出预设

**格式: H.264**
```
基本视频设置：
├── 宽度: 1920
├── 高度: 1080
├── 帧速率: 60 fps
├── 场序: 逐行
├── 长宽比: 方形像素 (1.0)
└── 性能: 以最大深度渲染

比特率设置：
├── 比特率编码: VBR，2次
├── 目标比特率: 15 Mbps
├── 最大比特率: 20 Mbps
└── 关键帧距离: 自动

音频设置：
├── 音频格式: AAC
├── 采样率: 48000 Hz
├── 声道: 立体声
└── 比特率: 320 kbps
```

### 7.2 文件命名规范

```
格式：[类型]_[日期]_[标题]_[版本]

示例：
- [测评]_0317_用了10款翻译插件都不满意_v1.mp4
- [教程]_0319_正式上架NotOnlyTranslator完整评测_final.mp4
- [直播]_0330_你的英语阅读问题我来答_replay.mp4
```

## 十、附录

### 10.1 B站创作中心数据解读

**关键指标：**
- **完播率**: >40% 为优秀，影响推荐权重
- **互动率**: (点赞+投币+收藏+评论)/播放量
- **粉丝转化**: 关注/播放 >2% 为优质内容

**优化方向：**
- 完播率低 → 缩短视频/增强节奏
- 互动率低 → 增加互动引导/设置争议点
- 转化率低 → 片尾加强CTA

### 10.2 常用快捷键 (Premiere)

```
C - 剃刀工具
V - 选择工具
Q/W - 波纹修剪上/下一个编辑点
K - 剪切
空格 - 播放/暂停
J/K/L - 反向播放/暂停/正向播放
,/. - 逐帧后退/前进
M - 添加标记
Ctrl+K - 剪切所有轨道
```
