## 4. Apple Podcasts 提交

### 4.1 RSS Feed 准备

**Feed 技术要求：**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<rss xmlns:itunes="http://www.itunes.com/dtds/podcast-1.0.dtd" version="2.0">
  <channel>
    <title>NotOnlyTranslator 英语学习电台</title>
    <link>https://notonlytranslator.com</link>
    <language>zh-cn</language>
    <copyright>Copyright 2026 NotOnlyTranslator</copyright>
    <itunes:author>NotOnlyTranslator Team</itunes:author>
    <itunes:summary>智能翻译插件 NotOnlyTranslator 官方出品的学习类播客</itunes:summary>
    <description>为英语学习者打造的轻知识播客，分享高效学习方法、语言科技趋势和真实用户成长故事。</description>
    <itunes:owner>
      <itunes:name>NotOnlyTranslator</itunes:name>
      <itunes:email>hi@notonlytranslator.com</itunes:email>
    </itunes:owner>
    <itunes:image href="https://notonlytranslator.com/assets/podcast-cover-3000.jpg"/>
    <itunes:category text="Education">
      <itunes:category text="Language Learning"/>
    </itunes:category>
    <itunes:explicit>clean</itunes:explicit>
    <!-- Episodes -->
  </channel>
</rss>
```

**必需字段：**
- 标题、描述、作者、分类
- 封面图片（1400x1400 至 3000x3000）
- 语言设置（zh-cn）
- 版权信息

### 4.2 提交流程

**方式：通过 Podcast Connect**
1. 访问 https://podcastsconnect.apple.com/
2. 使用 Apple ID 登录
3. 点击「+」添加新节目
4. 输入 RSS Feed URL
5. 等待审核（1-7个工作日）

**审核状态追踪：**
- 状态查看：Podcast Connect 后台
- 邮件通知：审核结果会邮件通知

### 4.3 Apple Podcasts 特有优化

**关键词优化：**
- 标题中包含关键词：英语学习、学习方法
- 描述中自然融入关键词

**多语言支持（可选）：**
- 添加英文版 Shownotes
- 设置语言标签

## 9. UTM 参数与数据追踪设置

### 9.1 UTM 参数规范

**参数结构：**
```
?utm_source={平台}&utm_medium=podcast&utm_campaign={活动}&utm_content={期数}
```

**平台参数值：**

| 平台 | utm_source | utm_medium |
|------|------------|------------|
| 小宇宙 | xiaoyuzhou | podcast |
| 喜马拉雅 | ximalaya | podcast |
| Apple Podcasts | apple_podcasts | podcast |
| Spotify | spotify | podcast |
| 网易云音乐 | netease | podcast |
| 荔枝FM | lizhi | podcast |

**示例链接：**
```
https://notonlytranslator.com?utm_source=xiaoyuzhou&utm_medium=podcast&utm_campaign=S01E01&utm_content=trailer
```

### 9.2 各平台链接配置

**小宇宙：**
- 官网链接带 UTM
- Shownotes 中的链接带 UTM

**喜马拉雅：**
- 专辑介绍中的官网链接
- 每期节目的详情页链接

**Apple/Spotify：**
- RSS Feed 中的链接
- 节目描述中的链接

### 9.3 数据追踪工具

**Google Analytics 4：**
- 创建播客流媒体专属事件
- 追踪播客带来的流量

**UTM 参数生成工具：**
```javascript
function generatePodcastURL(platform, episode, campaign = 'organic') {
  const baseURL = 'https://notonlytranslator.com';
  const params = new URLSearchParams({
    utm_source: platform,
    utm_medium: 'podcast',
    utm_campaign: campaign,
    utm_content: episode
  });
  return `${baseURL}?${params.toString()}`;
}

// 使用示例
generatePodcastURL('xiaoyuzhou', 'S01E01');
// 输出: https://notonlytranslator.com?utm_source=xiaoyuzhou&utm_medium=podcast&utm_campaign=organic&utm_content=S01E01
```

## 12. 风险管理

### 12.1 常见审核问题

| 问题 | 影响 | 解决方案 |
|------|------|----------|
| 名称与已有播客重复 | 需改名 | 添加副标题区分 |
| 分类选择不当 | 流量不精准 | 重新选择分类 |
| 封面不符合规范 | 审核不通过 | 按规格重新设计 |
| 简介含敏感词 | 审核不通过 | 修改文案重新提交 |
| 企业认证材料不全 | 审核延迟 | 准备完整材料 |

### 12.2 应急预案

**审核不通过：**
- 立即查看驳回原因
- 24小时内修改重新提交
- 备用方案：先用个人账号发布

**平台技术问题：**
- 联系平台客服
- 使用官方帮助中心
- 社群求助
