## 1. 预告片定位与目标

### 1.1 核心目标

- **介绍播客**: 让潜在听众快速了解这是什么播客、为谁而做
- **建立期待**: 激发听众对正式内容的兴趣和期待
- **引导订阅**: 明确的行动号召，鼓励立即订阅
- **品牌展示**: 传递专业、知性、温暖的品牌调性

### 1.2 目标听众

- 考研学生、外企白领、英语爱好者
- 正在寻找高效英语学习方法的人
- NotOnlyTranslator 插件现有用户

### 1.3 成功指标

| 指标 | 目标值 | 追踪方式 |
|------|--------|----------|
| 完播率 | >70% | 小宇宙后台 |
| 订阅转化率 | >15% | 订阅增长/播放量 |
| 评论数 | >50条（首周） | 平台后台 |

## 2. 预告片内容规划

### 2.1 时长与结构

**总时长**: 1分30秒 - 2分钟

```
00:00-00:20  片头音乐 + 品牌音效标识
00:20-00:45  开场钩子 + 痛点共鸣
00:45-01:15  播客介绍 + 内容预览
01:15-01:40  主持人介绍 + 品牌背书
01:40-01:55  行动号召 + 订阅引导
01:55-02:00  片尾音乐淡出
```

### 2.2 脚本内容

#### 开场钩子 (00:20-00:45)

```
[音乐渐入，主持人声音温暖、亲和]

"你有没有遇到过这样的情况——
读一篇英文文章，每个单词都认识，但连在一起就不知所云？
或者，依赖全文翻译，看的时候很爽，
但关掉页面，发现自己什么也没记住？

如果你点头了，那这个节目，就是为你准备的。"
```

#### 播客介绍 (00:45-01:15)

```
"欢迎来到 NotOnlyTranslator 英语学习电台。

在这里，我们不谈枯燥的语法规则，
也不教你死记硬背单词的方法。
我们分享的是——
真正有效的英语学习策略，
让你能在自然阅读中，不知不觉提升词汇量。

每一期节目，
我们会探讨一个学习方法论，
或是分享一位学习者的真实成长故事，
或是带你实战练习高效阅读技巧。"
```

#### 主持人介绍 (01:15-01:40)

```
"我是你的学习伙伴 [主持人名字]，
也是 NotOnlyTranslator 的创始人/团队成员。

过去几年，我见证了成千上万的学习者
从「害怕英文阅读」到「享受英文阅读」的转变。
现在，我想把这些经验和洞察，通过声音传递给你。"
```

#### 行动号召 (01:40-01:55)

```
"这档播客每周一更新，
每期15-25分钟，正好适合你的通勤或午休时间。

现在就点击订阅按钮，
让我们下周见。

[音乐渐强]

NotOnlyTranslator 英语学习电台——
让英语学习，更简单。"
```

## 3. 录制要求

### 3.1 录音环境

参照 [podcast-recording-setup-spec.md](./podcast-recording-setup-spec.md) 执行：

- 安静环境（底噪 <-60dB）
- 麦克风距离 15-20cm
- 试录3次，选择最佳一版

### 3.2 语气与风格

| 要素 | 要求 |
|------|------|
| 语速 | 适中偏慢（预告片需要清晰传达信息） |
| 语气 | 温暖、亲和、有温度，但不失专业感 |
| 停顿 | 关键句后有1-2秒停顿，让信息沉淀 |
| 情绪 | 积极、期待、真诚 |

### 3.3 音频质量标准

- 采样率：44.1kHz
- 比特率：320kbps
- 格式：WAV（原始）/ MP3（发布）
- 响度：-16 LUFS
- 无喷麦、无齿音过重、无环境杂音

## 4. 后期制作清单

### 4.1 处理流程

```
1. 降噪处理
   □ 去除背景杂音
   □ 去除呼吸声（保留自然呼吸）

2. 剪辑
   □ 去除口误、过长停顿
   □ 保持自然流畅的对话感

3. 音量调整
   □ 人声标准化至 -16 LUFS
   □ 确保音量一致，无忽大忽小

4. 音乐混入
   □ 片头音乐：18秒处主持人声音淡入（0.5秒）
   □ 片尾音乐：主持人结束后1秒淡入，最后3-5秒淡出
   □ 音乐音量比人声低 3-6dB

5. 最终质检
   □ 完整试听一遍
   □ 检查是否有技术问题
   □ 导出最终版本
```

### 4.2 交付文件

```
podcast-trailer/
├── NOT_trailer_v1.wav      # 无损版本（存档）
├── NOT_trailer_v1.mp3      # 发布版本（320kbps）
└── NOT_trailer_script.pdf  # 最终版脚本（归档）
```

## 7. 时间安排

### 7.1 制作时间线

| 阶段 | 时长 | 行动 |
|------|------|------|
| 脚本定稿 | Day 1 | 确定最终脚本，选择钩子版本 |
| 录制 | Day 2 | 完成预告片录音 |
| 后期制作 | Day 3 | 剪辑、混音、加音乐 |
| 审核修改 | Day 4 | 内部试听，收集反馈，修改 |
| 发布准备 | Day 5 | 准备 Shownotes、封面、推广素材 |
| 正式上线 | Day 6 | 多平台发布 + 社媒推广 |

### 7.2 与正式节目的衔接

```
预告片上线（周一）
    ↓ 间隔3-4天
第一期节目预告（周四/五，社交媒体）
    ↓ 间隔3-4天
第一期正式上线（下周一）
```

## 8. 风险评估与应对

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 录制效果不理想 | 延期 | 预留2天重录时间 |
| 后期修改次数多 | 延期 | 明确一次性修改范围 |
| 发布平台审核慢 | 延期 | 提前1天提交审核 |
| 初期播放量低 | 士气 | 设定合理预期，专注长期价值 |
