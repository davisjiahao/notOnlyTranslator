## 1. 设计概念说明

### 1.1 视觉理念

**核心概念**：「翻译有声，学习有形」

播客封面需要传达 NotOnlyTranslator 品牌的核心价值——智能、专业、陪伴。通过视觉符号将「英语学习」与「音频媒介」自然融合，让目标听众在缩略图尺寸下也能快速识别品牌和内容调性。

**设计关键词**：
- **智能**：科技感、现代化、简洁线条
- **陪伴**：温暖、亲和、易接近
- **专业**：可信、权威、有深度

### 1.2 品牌调性映射

| 品牌属性 | 视觉表达方式 |
|----------|-------------|
| 智能翻译 | 品牌Logo + 文字符号 |
| 英语学习 | 字母元素、对话气泡 |
| 音频媒介 | 耳机、声波、音符 |
| 科技感 | 渐变色、几何图形 |
| 专业可信 | 克制的配色、清晰的排版 |

### 1.3 情感目标

听众看到封面时应该产生：
- 「这是一个专业的英语学习节目」
- 「看起来很有科技感，应该有干货」
- 「设计简洁干净，值得关注」

## 2. 详细设计规格

### 2.1 尺寸规格

| 参数 | 数值 | 说明 |
|------|------|------|
| 画布尺寸 | 3000 x 3000 px | 正方形，适配所有平台 |
| 安全区域 | 2400 x 2400 px | 距边缘300px内不放核心元素 |
| 最小显示尺寸 | 100 x 100 px | 缩略图测试尺寸 |
| 文件格式 | PNG / JPG | 主文件PNG，备用JPG |
| 色彩模式 | sRGB | 网页标准色域 |
| 文件大小 | < 2MB | 上传限制 |

### 2.2 配色方案

**主色**：

| 色彩名称 | HEX | RGB | 用途 |
|----------|-----|-----|------|
| 品牌蓝 | #3B82F6 | 59, 130, 246 | 主背景渐变起点、核心元素 |
| 品牌紫 | #8B5CF6 | 139, 92, 246 | 渐变终点、装饰元素 |
| 纯白 | #FFFFFF | 255, 255, 255 | 文字、高亮、Logo |

**辅助色**：

| 色彩名称 | HEX | RGB | 用途 |
|----------|-----|-----|------|
| 浅蓝 | #DBEAFE | 219, 234, 254 | 背景、装饰光晕 |
| 深蓝 | #1E40AF | 30, 64, 175 | 文字阴影、强调 |
| 浅紫 | #EDE9FE | 237, 233, 254 | 装饰元素 |

**渐变规范**：
```
主渐变（背景）：
- 类型：线性渐变
- 方向：左上 → 右下（135deg）
- 起点：#3B82F6 (100%)
- 终点：#8B5CF6 (100%)
```

### 2.3 字体规范

**中文标题字体**：

| 层级 | 字体 | 字重 | 字号 | 行高 |
|------|------|------|------|------|
| 主标题 | 思源黑体 / 苹方 | Bold (700) | 420px | 1.2 |
| 副标题 | 思源黑体 / 苹方 | Medium (500) | 180px | 1.3 |

**备选字体方案**：
- 方案A：苹方（Apple生态优先）
- 方案B：思源黑体（跨平台一致）
- 方案C：阿里巴巴普惠体（免费商用）

**英文/数字字体**：
- 主用：Inter / SF Pro
- 备选：Roboto / Arial

**文字处理**：
- 标题文字需要轻微阴影增加层次感
- 阴影参数：`0 4px 20px rgba(0,0,0,0.15)`
- 确保在缩略图尺寸下仍清晰可读

### 2.4 边距与布局

**安全边距**：
```
┌────────────────────────────────────┐
│            300px (边距)             │
│    ┌─────────────────────────┐    │
│    │                         │    │
│300 │     核心内容区域         │300 │
│px  │     2400 x 2400 px      │px  │
│    │                         │    │
│    └─────────────────────────┘    │
│            300px (边距)             │
└────────────────────────────────────┘
```

**元素层级**（从下到上）：
1. 背景层：渐变填充
2. 装饰层：声波、光晕、几何图形
3. Logo层：品牌标识（居中偏上）
4. 文字层：播客名称（居中偏下）

## 3. 视觉元素清单

### 3.1 Logo使用规范

**品牌Logo规格**：

基于现有品牌图标（NT字母标识）进行播客版本适配：

| 元素 | 规格 |
|------|------|
| Logo尺寸 | 600 x 600 px（占画布20%） |
| Logo位置 | 画布中心偏上，Y轴偏移-150px |
| 圆角半径 | 90px（与原Logo风格一致） |
| 背景 | 纯白 #FFFFFF |
| Logo内文字 | "NT" 保持原有设计 |

**Logo处理**：
- 播客版Logo在原Logo基础上增加音频元素装饰
- 保留渐变背景：#3B82F6 → #8B5CF6
- 圆角矩形容器保持品牌一致性
- 白色"NT"文字居中

### 3.2 音频视觉元素

**声波图形**：

```
设计方向：简洁的声波线条

规格：
- 线条数量：3-5条
- 线条宽度：8-12px
- 线条高度：递增再递减的节奏感
- 颜色：白色，透明度 70%-90%
- 位置：Logo右侧或下方
- 动态感：呈现"播放中"的视觉暗示
```

**声波示意**：
```
     ▁ ▃ ▅ ▇ ▅ ▃ ▁
```

**耳机图形**（可选方案）：

```
设计方向：极简线条耳机图标

规格：
- 图标尺寸：120-150px
- 线条宽度：6-8px
- 颜色：白色
- 透明度：85%
- 位置：Logo左上角或右上角装饰
- 风格：圆角、现代、无多余细节
```

### 3.3 装饰元素

**背景装饰**：

| 元素 | 规格 | 用途 |
|------|------|------|
| 光晕圆 | 直径800px，纯白径向渐变，透明度5-10% | 增加背景层次 |
| 几何点阵 | 小圆点阵列，直径4-6px，间距30px，白色3%透明度 | 科技感纹理 |
| 曲线装饰 | 贝塞尔曲线，白色10%透明度，线宽2px | 动感元素 |

**装饰原则**：
- 装饰元素不抢主角（Logo和文字）的风头
- 保持画面呼吸感，不拥挤
- 装饰服务于品牌调性，不随意添加

## 4. 多平台适配说明

### 4.1 各平台显示尺寸

| 平台 | 显示尺寸 | 显示位置 | 关键注意事项 |
|------|----------|----------|-------------|
| **小宇宙** | 200x200px（列表）<br>300x300px（详情） | 节目封面、搜索结果 | 重点测试缩略图可读性 |
| **喜马拉雅** | 180x180px（列表）<br>300x300px（节目页） | 专辑封面、推荐位 | 浅色背景也需可见 |
| **Apple Podcasts** | 200x200px（列表）<br>300x300px（详情） | 节目封面、搜索 | 国际化风格，避免纯中文 |
| **Spotify** | 200x200px（列表）<br>300x300px（详情） | 节目封面、推荐 | 深色模式兼容性测试 |
| **网易云音乐** | 150x150px（列表）<br>300x300px（详情） | 节目封面、动态 | 年轻化审美 |

### 4.2 缩略图测试清单

**100x100px 缩略图测试标准**：

- [ ] Logo "NT" 清晰可辨
- [ ] 文字主体可读
- [ ] 配色对比度足够
- [ ] 整体图形轮廓清晰
- [ ] 无细小元素模糊

**测试方法**：
1. 将设计稿缩放到 100x100px
2. 在浅色背景（#F5F5F5）上查看
3. 在深色背景（#1A1A1A）上查看
4. 在不同设备屏幕上查看（手机/平板/电脑）

### 4.3 深色模式兼容性

**Spotify / 网易云音乐深色模式适配**：

- 确保白色文字在深色背景上有足够对比
- 避免使用纯黑元素（在深色模式会消失）
- 测试渐变背景在深色界面中的表现
- 边框/描边元素需考虑深色模式可见性

### 4.4 特殊场景适配

| 场景 | 适配要求 |
|------|----------|
| 圆形裁剪 | 核心元素在 2400px 圆形区域内居中 |
| 带圆角矩形裁剪 | 模拟小宇宙/喜马拉雅的圆角展示效果 |
| 阴影叠加 | 测试在有阴影的平台上的显示效果 |
| 白色边框 | 部分平台自动添加白色边框，预留视觉空间 |

## 6. 设计参考与灵感

### 6.1 风格参考

**类似播客封面参考**：
- 中文播客：得意忘形、随机波动、忽左忽右
- 英语学习：潘吉芬尼根、英语流利说
- 科技类：科技早知道、硅谷101

**设计风格关键词**：
- Minimalist（极简）
- Modern Tech（现代科技）
- Clean Typography（干净排版）
- Gradient Background（渐变背景）

### 6.2 设计原则

1. **少即是多**：避免元素堆砌，每个元素都要有意义
2. **缩略图优先**：设计时以100px尺寸作为检验标准
3. **品牌一致**：与产品图标风格统一
4. **跨文化考量**：兼顾国内外平台，避免纯文化符号

## 7. 设计验收标准

### 7.1 视觉验收

- [ ] Logo清晰可辨，在100px尺寸下仍可识别
- [ ] 文字可读性高，无模糊或锯齿
- [ ] 配色符合品牌规范
- [ ] 渐变过渡平滑，无断层
- [ ] 边缘处理干净，无白边或杂色

### 7.2 技术验收

- [ ] 尺寸精确为3000x3000px
- [ ] 文件大小<2MB
- [ ] 色彩模式为sRGB
- [ ] PNG格式无压缩损失
- [ ] JPG格式质量>85%

### 7.3 平台测试

- [ ] 小宇宙显示正常
- [ ] 喜马拉雅显示正常
- [ ] Apple Podcasts显示正常
- [ ] Spotify显示正常（含深色模式）
