## 视觉规格

### Feed 帖子
- **尺寸**: 1080 x 1080 px (正方形)
- **格式**: JPG 或 PNG
- **色彩模式**: RGB
- **DPI**: 72

### Stories
- **尺寸**: 1080 x 1920 px (9:16)
- **格式**: JPG 或 PNG
- **色彩模式**: RGB
- **DPI**: 72

### Reels 封面
- **尺寸**: 1080 x 1920 px (9:16)
- **格式**: JPG
- **安全区域**: 顶部和底部各留 250px 空白 (避免被 UI 遮挡)
