# LinkedIn 内容视觉模板设计规范

## 设计系统概述

### 品牌色彩
```
主色：#3B82F6（科技蓝）
辅助色：#10B981（成功绿）
强调色：#F59E0B（警示橙）
背景色：#F8FAFC（浅灰白）
文字色：#1F2937（深灰）
```

### 字体规范
```
标题字体：Inter Bold / PingFang SC Bold
正文字体：Inter Regular / PingFang SC Regular
等宽字体：JetBrains Mono（用于代码/英文单词）

字号层级：
- 主标题：32px
- 副标题：24px
- 正文：18px
- 小字：14px
```

## 模板类型清单

### 模板 A：产品对比图 (1200×800)
**用途**：发布贴 #1（D-Day 发布）的配图

**布局**：
```
┌─────────────────────────────────────┐
│  [左：原文]      │  [右：翻译效果]   │
│                  │                   │
│  英文技术文档    │  高亮生词 + 翻译  │
│  截图            │  悬浮提示         │
│                  │                   │
└─────────────────────────────────────┘
```

**设计要点**：
- 左图：真实英文网页截图（如 Medium 技术文章）
- 右图：NotOnlyTranslator 高亮效果
- 底部添加品牌 Logo 和 Chrome Web Store badge
- 左右对比用箭头连接，突出变化

**文件命名**：`template-01-product-comparison.psd`

### 模板 B：Carousel 卡片 (1080×1350)
**用途**：发布贴 #2（功能亮点）8 页 Carousel

**统一元素**：
- 页码标记：1/8, 2/8...（右上角，24px，主色）
- 品牌 Logo：每页底部居中，40px 高度
- 进度指示器：底部进度条

**各页内容**：

| 页码 | 背景 | 标题 | 内容 |
|-----|------|------|------|
| 1 | 渐变蓝 | 为什么同样的文章... | 问题引入 |

| 3 | 浅蓝 | 我们换个思路 | 解决方案概述 |
| 4 | 白色 | NotOnlyTranslator 如何工作 | 3 步骤流程图 |
| 5 | 浅灰 | 点击生词 → 即时翻译 | 功能演示截图 |
| 6 | 白色 | 你的词汇量，你做主 | 3 个操作按钮样式 |
| 7 | 渐变灰 | 谁需要这个工具？ | 4 类用户头像+描述 |
| 8 | 主色蓝 | CTA 页面 | 下载引导 |

**设计文件**：`template-02-carousel-01.psd` ~ `template-02-carousel-08.psd`

## 视觉风格指南

### 图片风格
- **截图**：使用真实网页截图，添加圆角和阴影
- **图标**：统一使用 Fluent Emoji 或 Notion 风格图标
- **配色**：以蓝白为主，避免过多颜色
- **留白**：每页至少 20% 留白，避免拥挤

### 品牌一致性检查
- [ ] Logo 位置和大小一致
- [ ] 主色使用不超过 60% 画面
- [ ] 字体不超过 2 种
- [ ] 所有图片导出前压缩优化
