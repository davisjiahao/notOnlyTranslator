## 一、百度知道回答模板

### 1.1 翻译插件推荐类

**目标问题**：
- "翻译插件哪个好用？"
- "Chrome有什么好用的翻译扩展？"
- "浏览器翻译插件推荐"

**回答模板**：

```
作为一个长期使用各种翻译工具的人，我来分享一下我的使用心得。

目前市面上主流的翻译插件主要有几款：

1️⃣ 沉浸式翻译 - 特点是双语对照显示，适合想要对照学习的用户
2️⃣ NotOnlyTranslator - 特点是只翻译生词，智能分级，非常适合英语学习者
3️⃣ 彩云小译 - 整句翻译流畅，适合快速浏览

如果你主要是【考研/四六级备考】，推荐试试 NotOnlyTranslator，它可以
根据你的水平智能判断哪些词需要翻译，还能自动记录生词复习。

如果只是【日常浏览】，沉浸式翻译或彩云小译都不错。

建议可以先安装试用，找到自己用着顺手的。

注意：Chrome插件需要科学上网才能安装，Edge浏览器可以直接从
微软商店安装部分插件。
```

### 1.2 英语学习方法类

**目标问题**：
- "如何提高英语阅读速度？"
- "英语单词怎么记最快？"
- "考研英语阅读有什么技巧？"

**回答模板**：

```
提高英语阅读速度，核心在于【减少查词中断】+【培养猜词能力】。

分享几个实用的方法：

📖 方法一：分级阅读
找适合自己水平的材料，遇到生词不要立刻查，先尝试根据上下文猜。
如果理解不了整句，再查关键词。

📖 方法二：使用智能翻译工具
推荐使用 NotOnlyTranslator 这类插件，它会根据你的词汇量，
只翻译你真正不认识的单词，这样阅读更流畅。

📖 方法三：大量输入
每天保持30分钟以上的英语阅读，材料可以是新闻、博客、技术文档等
自己感兴趣的内容。

📖 方法四：词汇积累
准备一个生词本，记录阅读中遇到的生词，定期复习。
有些工具（如上面提到的插件）可以自动帮你做这个工作。

坚持练习，阅读速度会逐渐提升的。
```

### 1.3 程序员英语类

**目标问题**：
- "程序员怎么提高英语水平？"
- "如何高效阅读英文技术文档？"
- "Stack Overflow英文看不懂怎么办？"

**回答模板**：

```
程序员学英语，最好的方式就是【在工作中学习】。

💡 建议1：直接读官方文档
不要一上来就找中文翻译，试着直接看英文文档。遇到不懂的词再查，
技术文档的词汇其实重复率很高，看多了自然就熟了。

💡 建议2：善用工具辅助
推荐使用 Chrome 的 NotOnlyTranslator 插件，它针对程序员做了优化，
只翻译你不认识的技术术语，不会破坏代码阅读体验。

💡 建议3：从熟悉的领域开始
先读你熟悉的技术的英文文档，这样即使有些词不认识，也能猜出大意。

💡 建议4：积累技术词汇
技术英语和普通英语有些不同，建议单独整理一份技术词汇表。

💡 建议5：多用英文搜索引擎
养成用 Google + 英文关键词搜索的习惯，结果质量往往更好。

技术文档的英语其实不难，主要是术语多。坚持半年，你会发现
英文文档也没那么可怕了。
```

### 2.1 工具使用教程

**标题格式**：
- "Chrome浏览器翻译插件安装使用教程"
- "如何用XXX插件提升英语阅读效率"
- "NotOnlyTranslator翻译插件新手入门"

**文章结构**：

```markdown
## 工具/原料
- Chrome浏览器（或Edge）
- NotOnlyTranslator插件
- OpenAI API Key（可选）

## 方法/步骤

### 第一步：安装插件
1. 打开 Chrome 网上应用店
2. 搜索 "NotOnlyTranslator"
3. 点击"添加至Chrome"

### 第二步：基础设置
1. 点击浏览器右上角插件图标
2. 选择"选项"进入设置
3. 设置你的英语水平（四六级/考研/雅思等）

### 第三步：开始使用
1. 打开任意英文网页
2. 插件会自动标注生词
3. 鼠标悬停即可查看翻译

## 注意事项
- 首次使用需要配置API（或使用默认配置）
- 可以在设置中调整词汇量等级
- 生词会自动保存到词汇本

## 适用场景
- 考研英语阅读
- 四六级备考
- 程序员技术文档阅读
- 日常英文新闻浏览
```

## 四、账号运营规范

### 4.1 养号策略

**第一阶段（1-2周）：新号养号**
- 每天浏览30分钟以上
- 回答5-10个非营销类问题
- 关注感兴趣的话题
- 完善个人资料

**第二阶段（3-4周）：轻度营销**
- 80%正常回答 + 20%软植入
- 避免每个回答都带产品
- 积累采纳率

**第三阶段（5周+）：稳定运营**
- 可以适度增加营销比例
- 但仍需保持回答质量
- 维护账号信誉

### 4.2 回答规范

**✅ 正确做法**：
- 真实、有用的回答
- 先解决问题，再自然提及产品
- 保持专业、友好的语气
- 适当使用表情符号增加亲和力
- 回答长度适中（100-300字）

**❌ 禁止做法**：
- 直接广告/链接
- 复制粘贴相同内容
- 明显的营销话术
- 短时间内大量回答
- 回答与问题无关

### 4.3 风险规避

- 不要同一个IP登录多个账号
- 不要使用明显的营销号特征
- 回答间隔不要太短
- 保持账号活跃度
- 遵守百度社区规则
