# NotOnlyTranslator 完整功能指南：智能翻译插件怎么用

> 更新时间：2026年3月 | 适用版本：v1.0+

## 快速了解 NotOnlyTranslator

**NotOnlyTranslator** 是一款专为英语学习者设计的 Chrome 翻译插件。与其他翻译工具不同，它不是整段翻译，而是**根据你的词汇量，只翻译你真正不认识的单词**。

**核心理念**：在阅读中自然积累词汇，而不是依赖翻译。

---

## 核心功能一览

### 1. 智能分级翻译

**功能说明**：
- 根据你的词汇量水平（四六级/考研/雅思/托福/自定义），自动判断哪些单词需要翻译
- 只标注超出你当前水平的单词，保持阅读流畅性
- 支持整句模式和单词模式切换

**使用场景**：
- 考研英语真题阅读：只翻译超纲词，认识的词不打扰
- 技术文档阅读：专业术语翻译，常见词汇跳过
- 英文新闻浏览：根据你的水平动态调整

**如何设置**：
1. 点击浏览器工具栏的 NotOnlyTranslator 图标
2. 在"词汇量设置"中选择你的水平
3. 或进行"词汇量测试"获得精准评估

---

### 2. 词汇量评估测试

**功能说明**：
- 基于统计学算法的词汇量测试
- 测试过程约 3-5 分钟
- 结果用于智能分级翻译的基准

**测试原理**：
采用自适应测试算法，根据你的答题情况动态调整题目难度，快速收敛到真实词汇量水平。

**查看方式**：
- Popup 面板显示当前评估词汇量
- Options 页面查看详细词汇分布

---

### 3. 生词本管理

**功能说明**：
- 自动记录阅读中遇到的生词
- 支持手动添加/删除单词
- 分类管理：标记为"已掌握"或"需复习"

**操作方式**：
1. 点击网页上标注的高亮单词
2. 在弹出的 Tooltip 中点击"加入生词本"
3. 或点击"已掌握"从生词本移除

**数据同步**：
- 生词本保存在 Chrome 本地存储
- 支持导出为 JSON/CSV 格式

---

### 4. 闪卡复习模式

**功能说明**：
- 基于艾宾浩斯遗忘曲线的复习算法
- 支持键盘快捷键操作（空格翻转，数字键评分）
- 多种复习模式：新词学习、错题复习、随机抽查

**复习算法**：
- 新词：间隔 1 天后首次复习
- 认识：间隔指数增长（2天→4天→8天...）
- 不认识：回到 1 天间隔

**进入方式**：
1. 点击浏览器图标打开 Popup
2. 点击"开始复习"按钮
3. 或使用快捷键 Alt+Shift+R

---

### 5. 学习统计仪表盘

**功能说明**：
- 今日学习数据：新词数量、复习数量、学习时长
- 累计数据：总词汇量、掌握单词数、学习天数
- 词汇增长趋势图
- 学习热力图

**查看方式**：
- Popup 面板显示今日概览
- Options 页面查看完整统计

**数据维度**：
- 每日新词数
- 每日复习数
- 词汇量增长曲线
- 连续学习天数

---

### 6. 多翻译引擎支持

**支持的翻译服务**：

| 引擎 | 特点 | 配置难度 |
|------|------|----------|
| **OpenAI** | 翻译质量高，成本低 | 需 API Key |
| **Anthropic Claude** | 学术文本表现好 | 需 API Key |
| **自定义 API** | 支持任何 OpenAI 兼容接口 | 需配置端点 |

**配置方法**：
1. 进入 Options 设置页面
2. 选择"翻译服务"标签
3. 选择引擎并填入 API Key
4. 点击"测试连接"验证

**成本参考**（OpenAI GPT-4o-mini）：
- 平均每页翻译成本：¥0.01-0.03
- 轻度使用（每天 10 页）：月成本约 ¥10

**可选主题**：
- **浅色模式**：适合白天使用
- **深色模式**：适合夜间阅读

**三种模式**：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **学习模式** | 标注所有超纲词，显示翻译 | 精读学习 |

**快捷切换**：

- 或使用快捷键 Alt+Shift+M

## 常见问题

### Q: 为什么有些单词没有翻译？

**A**: 这是 NotOnlyTranslator 的核心设计理念。插件根据你的词汇量水平，只翻译"超纲"的单词。如果你发现认识的单词也被翻译了，可以在 Tooltip 中标记为"已掌握"，系统会学习你的实际水平。

### Q: 需要付费吗？

**A**: 插件本身完全免费。但翻译功能需要调用 LLM API（OpenAI/Anthropic），这需要你自己提供 API Key 并按使用量付费。轻度使用成本约每月 ¥10-20。

### Q: 支持哪些浏览器？

**A**: 目前支持 Chrome 和 Edge 浏览器。Firefox 版本正在开发中。

### Q: 生词本数据会丢失吗？

**A**: 数据存储在浏览器本地，清理浏览器数据会导致丢失。建议定期使用导出功能备份。

### Q: 如何反馈问题或建议？

**A**: 可以通过以下渠道反馈：
- Chrome Web Store 评价
- GitHub Issues（开源项目）
- 官方社交媒体账号

## 最佳实践建议

### 对于初学者（词汇量 < 4000）
1. 从四级词库开始
2. 每天阅读 1-2 篇简单文章
3. 认真复习每天的生词
4. 每周做一次词汇量测试

### 对于备考党（考研/四六级）
1. 选择对应考试的词库
2. 重点做真题阅读
3. 精读每篇文章，不追求速度
4. 利用复习功能巩固记忆

### 对于进阶学习者（词汇量 > 6000）
1. 使用雅思/托福词库
2. 尝试流畅模式阅读
3. 阅读英文原版书籍
4. 关注熟词僻义的积累
