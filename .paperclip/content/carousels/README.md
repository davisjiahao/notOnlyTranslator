## Best Practices for Designers

1. **Follow the specs** in each file exactly
2. **Keep text minimal** - max 20 words per slide
3. **Use high contrast** for readability
4. **Maintain brand colors** consistently
5. **Include swipe indicators** on cover slides
6. **End with clear CTA** on final slides
