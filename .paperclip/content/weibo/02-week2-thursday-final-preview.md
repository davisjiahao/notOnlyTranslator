---

🎯 核心功能：智能分级翻译

这是产品的灵魂。

不同于其他翻译工具的"全文翻译"，
NotOnlyTranslator 会：

✓ 评估你的词汇量水平
✓ 智能识别超出你能力的单词
✓ 只翻译那些"你应该不认识"的词
✓ 保持阅读的流畅性

举例：
一篇技术文章有100个单词，
如果你的水平是8000词汇量，
系统可能只翻译其中8-10个生词。

其余90个单词保持英文，
让你在阅读中保持英语思维。

---

📚 辅助功能一：词汇管理系统

每个被翻译的单词，
都会被记录到你的个人词汇库。

你可以：
✓ 查看所有查过的生词
✓ 标记"已掌握"或"需复习"
✓ 按时间、难度、领域筛选
✓ 导出词汇表（Anki/Excel格式）

这不再是"查完就忘"，
而是有迹可循的学习积累。

---

📊 辅助功能二：学习统计

系统会追踪你的学习数据：

✓ 每日阅读时长
✓ 每周新增词汇
✓ 词汇量增长曲线
✓ 各领域阅读占比

用数据可视化你的进步，
让学习过程有明确的正反馈。

---

⚙️ 个性化设置

你可以根据自己的习惯定制：

✓ 目标词汇量（8000/10000/12000/自定义）
✓ 翻译显示方式（下划线/高亮/悬浮）
✓ 词典来源（内置/DeepL/OpenAI）
✓ 复习提醒频率

让工具适应你，
而不是你去适应工具。

---

🔒 隐私保护

所有数据都存储在本地，
不会上传到任何服务器。

你的阅读历史、词汇记录、学习数据，
完全属于你自己。

---

📅 上线信息

正式发布：2026年4月2日 10:00

价格：免费版（基础功能）+ 高级版（完整功能）

终身会员免费（含所有高级功能）
