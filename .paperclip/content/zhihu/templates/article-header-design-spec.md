# 知乎文章头图设计规范

## 设计理念

**视觉风格**：专业、简洁、科技感 + 温度感
**配色方案**：品牌蓝 #3B82F6 + 辅助绿 #10B981 + 白色/灰色
**字体选择**：思源黑体 / PingFang SC

---

## 模板一：对比型（适用于测评类文章）

### 适用场景
- 工具对比测评
- 前后效果展示
- 功能对比

### 布局结构
```
┌─────────────────────────────────────┐
│                                     │
│    [左侧：Before/传统方式]            │
│              VS                     │
│    [右侧：After/NotOnlyTranslator]   │
│                                     │
│         文章标题（底部居中）          │
│                                     │
└─────────────────────────────────────┘
```

### 设计规格
- 尺寸：900 x 500px
- 背景：浅灰色渐变 #F3F4F6 → #E5E7EB
- 对比色：
  - Before：灰色 #9CA3AF
  - After：品牌蓝 #3B82F6
- 字体：标题 32px，加粗

## 首篇文章头图设计需求

### 文章信息
- **标题**：用了10款翻译插件后，我决定自己做一款
- **类型**：创业故事 + 测评
- **建议模板**：模板二（故事型）

### 设计要点
1. 背景使用深蓝色渐变，体现专业感
2. 文字使用白色，突出标题
3. 右下角放置 NotOnlyTranslator Logo（小尺寸）
4. 左侧添加装饰性图形：多个浏览器窗口的简化图标（代表多款插件）
5. 右侧添加一个突出的窗口图标（代表 NotOnlyTranslator）

### 文案内容
- 主标题：用了10款翻译插件后
- 副标题：我决定自己做一款
- 小字标签：英语学习效率工具
