## 这份词汇表有什么特别？

### 1. 场景化解释

不是字典式的学术定义，而是程序员能看懂的大白话：

> **deploy** (部署)
>
> 就像把做好的菜品端上桌给客人吃——你的代码开发完成，需要"端"到服务器上，让用户可以使用。这个过程就是 deploy。

### 2. 词根记忆法

告诉你单词的构成规律，记住一个就能推断出一串：

> **-ify 后缀**：表示"使...化"
> - simplify (简化) = simple + ify
> - verify (验证) = ver + ify
> - certify (认证) = cert + ify
> - qualify (限定/使合格) = qual + ify

### 3. 配套练习

每个词汇都配有来自真实技术文档的例句：

> **The API is deprecated. Use the new endpoint instead.**
>
> 这个 API 已被废弃，请使用新的端点。
