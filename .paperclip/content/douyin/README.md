## 🎬 视频制作规格

### 技术要求

| 元素 | 规格 |
|------|------|
| 宽高比 | 9:16 (竖屏) |
| 分辨率 | 1080x1920 最低 |
| 帧率 | 30fps 最低 |
| 长度 | 15-60秒（最佳: 21-34秒） |
| 字幕 | 始终包含，字体大而清晰 |
| 封面 | 自定义封面配合清晰钩子文字 |

### 品牌指南

**视觉识别：**
- **主色**: #3B82F6 (蓝色) - 信任、科技
- **辅色**: #10B981 (绿色) - 成长、学习
- **背景**: 简洁白色或微妙渐变
- **文字**: 粗体、高对比度、易读
- **Logo**: 角落水印（微妙）

**语气：**
- 友好且鼓励
- 从不居高临下地评价英语水平
- 庆祝进步，不追求完美
- 社区优先语言（"我们"、"一起"）

## 🏷️ 话题标签策略

### 推荐组合

每篇内容使用：2-3个 Tier 1 + 3-4个 Tier 2 + 2-3个 Tier 3 + 1-2个品牌标签

**Tier 1 - 高流量（亿级播放）：**

## 📝 内容类型说明

### 问题/解决方案 (30%)
展示全文翻译的痛苦 vs 我们的解决方案
- 示例钩子："别再一字一句翻译了"
- CTA："链接在主页"

### 学习技巧 (25%)
高效的英语学习方法分享
- 示例钩子："这个 Chrome 插件改变了我的阅读方式"
- CTA："收藏备用"

### 共鸣场景 (20%)
关于学习英语的搞笑/共鸣内容
- 示例钩子："当你意识到自己一直读错了"
- CTA："评论是否共鸣"

### 进步展示 (15%)
使用前后的词汇量对比
- 示例钩子："我的英语阅读水平：第1天 vs 第30天"
- CTA："收藏这个旅程"

### 热点参与 (10%)
跟随热门话题/音乐创作内容
- 示例钩子：热门音乐
- CTA："自己试试"
