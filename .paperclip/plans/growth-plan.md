# NotOnlyTranslator 增长策略计划

## 计划时间
2026-03-15

## 执行摘要

NotOnlyTranslator 是一款面向英语学习者的 Chrome 浏览器插件，核心价值在于**根据用户水平智能翻译网页内容**，帮助用户在阅读中自然习得词汇。

本计划聚焦于**产品发布前的增长准备**和**发布后的用户获取策略**，重点包括 Chrome Web Store ASO 优化、用户引导流程优化、以及社交媒体渠道布局。

## 三、产品内置增长机制

### 3.1 用户引导 (Onboarding) 优化

**当前流程问题**：
1. 首次安装后缺乏明确指引
2. API Key 配置步骤繁琐
3. 英语水平设置缺乏紧迫感

**优化方案**：

```
安装 → 欢迎弹窗 → 选择水平 → API 配置 → 引导浏览示例网站 → 完成
    (3秒)      (10秒)      (30秒)      (60秒)          (30秒)
```

**关键改进点**：

| 步骤 | 改进 | 预期效果 |
|------|------|----------|
| 欢迎弹窗 | 3 秒自动展示 + "立即体验"按钮 | 提升初始激活率 |
| 水平选择 | 预设考试选项 + "智能测试"按钮 | 降低决策门槛 |
| API 配置 | 提供免费 API quota（试用） | 降低首次使用障碍 |
| 示例网站 | 内置 3 个英文网站快捷入口 | 快速体验价值 |

### 3.2 病毒式传播机制

#### 3.2.1 词汇成就系统

| 成就名称 | 条件 | 奖励 | 分享触发 |
|----------|------|------|----------|
| 词汇小白 | 标记第一个生词 | 解锁成就徽章 | 手动 |
| 每日学习 | 连续 3 天使用 | 成就徽章 + 分享按钮 | 手动 |
| 词汇达人 | 掌握 100 词 | 成就徽章 + 分享按钮 | 手动 |
| 坚持不懈 | 连续 30 天 | 实体奖励（待定） | 自动 |

**分享内容设计**：
```
📚 NotOnlyTranslator
我已坚持学习 7 天，掌握了 23 个新单词！
我的英语水平估计提升至 CET-4 (4500词)
🔗 一起开始学习吧！
```

#### 3.2.2 每周学习报告

- 每周一自动推送学习报告
- 包含：本周新词数、学习时长、水平变化
- 鼓励分享到社交媒体
- 格式：图片 > 文字（便于传播）

### 3.3 用户召回机制

| 召回场景 | 触发条件 | 召回策略 |
|----------|----------|----------|
| 新用户激活 | 安装 24h 未完成设置 | 推送通知 + 简化流程 |
| 沉睡用户 | 7 天未使用 | "你的词汇需要复习啦"通知 |
| 流失预警 | 14 天未使用 | 优惠券/奖励召回 |

```
DAU = 新增用户 × 首日留存率 × (1 - 流失率)
```
