## 三、元数据优化

### 3.1 应用名称 (45字符限制)

**主名称**：
```
NotOnlyTranslator - 智能分级翻译，只翻译你不会的词
```

**备选名称**：
- `NotOnlyTranslator - 英语阅读助手，智能生词翻译`
- `NotOnlyTranslator - 网页翻译 & 词汇学习`
- `NotOnlyTranslator - AI分级翻译插件`

### 3.2 简短描述 (132字符限制)

**主描述**：
```
根据你的英语水平智能翻译，只翻译超出你词汇量的单词。读英文不中断，自然积累词汇。考研/四六级/留学必备阅读工具。
```

**备选描述**：
```
智能识别你的英语水平，只翻译你不会的生词。告别全文翻译的低效学习，在真实阅读中自然提升词汇量。
```

### 3.3 详细描述 (Markdown格式)

```markdown
## NotOnlyTranslator - 你的智能英语阅读助手

读英文文章时，是不是经常这样？
- 遇到生词要切出去查词典，读完后上下文忘了
- 用全文翻译，确实不用查词了，但你根本没在读英文
- 试了各种翻译工具，要么太简单，要么太复杂

### 核心功能

**智能分级翻译**
- 根据你的英语水平自动判断哪些词需要翻译
- 只翻译你不会的，已掌握的词汇保持原文
- 在阅读中自然区分已知/未知词汇

**上下文保留**
- 原文和翻译同时显示，随时对照
- 点击即可查看翻译，不打断阅读节奏
- 悬浮窗设计，不遮挡页面内容

**词汇积累**
- 自动记录你查询过的生词
- 标记已掌握词汇，优化翻译策略
- 支持词汇导出，配合 Anki 等工具复习

**隐私保护**
- 使用你自己的 API Key，数据完全私有
- 支持本地 LLM，无需联网
- 无追踪，无广告

### 适合谁用

- **考研党**：真题阅读、外刊精读
- **四六级考生**：提升阅读速度和词汇量
- **留学预备**：学术论文、申请资料阅读
- **职场人士**：技术文档、商务邮件
- **英语爱好者**：原版书籍、新闻网站

### 如何使用

1. 安装扩展后点击图标打开设置
2. 配置你的 OpenAI / Anthropic API Key
3. 设置你的英语水平（CET-4/CET-6/考研/雅思等）
4. 访问任意英文网页，自动生效

### 用户评价

> "考研阅读神器，再也不用来回查词了" - 用户A
> "比全文翻译好用太多，真正在学英语" - 用户B
> "界面简洁，翻译准确，强烈推荐" - 用户C

### 联系我们

- 问题反馈：[GitHub Issues]
- 功能建议：[邮箱/论坛]
- 使用教程：[文档链接]

---

**NotOnlyTranslator - 只翻译你不会的词，让英语阅读更自然**
```

## 四、视觉素材

| 尺寸 | 用途 | 要求 |

| 128x128 | 商店展示 | 高清PNG，无边框 |
| 48x48 | 扩展图标 | 清晰可辨识 |

**设计规范**：
- 使用品牌蓝色 (#3B82F6)
- 简洁现代风格
- 在深色/浅色背景都清晰
- 避免文字，使用图形符号

**截图策略**：

| 序号 | 场景 | 内容 |
|------|------|------|
| 1 | 首页展示 | 扩展图标 + 简洁介绍 |

| 3 | 设置页面 | 水平选择 + API配置 |
| 4 | 词汇管理 | 生词本界面 |
| 5 | 学习统计 | 数据统计仪表盘 |

**截图规范**：
- 统一使用 Chrome 深色主题
- 英文测试网站（BBC/CNN/技术博客）

- 保持视觉风格一致

## 五、评分与评论策略

### 5.1 评分目标

- 发布首周：4.5+
- 30天：4.6+
- 90天：4.7+

### 5.2 评论获取策略

**种子用户期 (D1-D7)**：
- 邀请内测用户优先评分
- 在产品内添加强引导（设置完成页）
- 目标：20条评论

**冷启动期 (D8-D30)**：
- 在B站/小红书视频描述中添加"好评有礼"
- 邮件回访活跃用户
- 目标：50条评论

**成长期 (D31+)**：
- 定期在更新日志中引导评分
- 解决负面反馈，邀请修改评价
- 目标：持续增长

### 5.3 负面评价处理

**处理原则**：
1. 24小时内回复所有负面评价
2. 真诚道歉，说明改进计划
3. 邀请用户测试修复版本
4. 问题解决后请求更新评价
