# NotOnlyTranslator 小红书营销执行计划

## 计划概述

**产品**: NotOnlyTranslator - 智能英语学习浏览器插件
**目标平台**: 小红书 (Xiaohongshu)
**目标用户**: 18-30岁英语学习者、留学生、考研党、职场人士
**计划周期**: 产品发布前 2 周准备 + 发布后 4 周运营
**目标**: 种草笔记 20+ 篇，累计曝光 10万+，引流 Chrome Web Store 1000+ 安装

## 三、视觉规范

### 3.1 封面设计标准

**尺寸**: 1080×1440px (3:4 竖版)

**封面类型**:
| 类型 | 适用场景 | 设计要点 |
|------|----------|----------|
| 产品截图型 | 功能介绍 | 高清UI截图 + 功能标注 |
| 对比型 | 测评内容 | Before/After 分屏展示 |
| 文字型 | 干货教程 | 大字标题 + 简洁背景 |
| 场景型 | 使用体验 | 真实使用场景照片 |

**封面文字规范**:
- 标题字体：思源黑体 Bold，字号 80-120px
- 副标题：字号 40-60px
- 颜色：主色 #1A1A1A，强调色 #FF2442（小红书红）
- 排版：左对齐或居中，不超过3行

### 3.2 配图规范

**产品截图要求**:
- 使用 Mac/Windows 高清截图（2x分辨率）
- 截图区域整洁，关闭无关标签页
- 标注关键功能点（使用箭头、圆框）
- 统一使用浅色/白色浏览器主题

**图片数量建议**:
- 好物分享：5-9张（多角度展示）
- 干货教程：3-6张（步骤分解）
- 对比测评：4-8张（对比表格+细节）

### 3.3 配色方案

**品牌色**:
- 主色：#4F46E5 (Indigo) - 代表学习、智慧
- 辅助色：#10B981 (Emerald) - 代表成长、进步
- 强调色：#F59E0B (Amber) - 用于重点标注
