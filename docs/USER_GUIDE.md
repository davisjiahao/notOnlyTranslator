## 快速入门指南

### 第一步：安装扩展

1. 打开 Chrome 浏览器，访问 Chrome Web Store
2. 搜索 "NotOnlyTranslator"
3. 点击"添加至 Chrome"
4. 在弹出的确认框中点击"添加扩展程序"

### 第二步：配置 API 密钥

**首次使用必须配置 API 密钥才能使用翻译功能。**

1. 点击浏览器工具栏上的 NotOnlyTranslator 图标
2. 点击"打开设置"
3. 选择"翻译提供商"标签
4. 选择您想要使用的翻译服务（OpenAI 或 Anthropic）
5. 输入对应的 API 密钥
6. 点击"保存"

> 💡 **还没有 API 密钥？** 请参考下方的 [API Key 设置教程](#api-key-设置教程)

### 第三步：设置英语水平

1. 在设置页面，切换到"一般"标签
2. 拖动"英语水平"滑块选择您的当前水平
   - **初级**：适合词汇量 < 2000 的学习者
   - **中级**：适合词汇量 2000-5000 的学习者
   - **高级**：适合词汇量 > 5000 的学习者
3. 保存设置

### 第四步：开始使用

1. 访问任意英文网站（如 Medium、BBC News、Wikipedia）
2. 稍等片刻，扩展会自动扫描页面内容
3. 超出您水平的单词会被高亮显示
4. 鼠标悬停在高亮单词上即可查看翻译
5. 点击"✓"标记为已掌握，点击"✗"标记为学习中

### 第五步：复习单词（可选）

1. 点击扩展图标，选择"单词复习"
2. 使用闪卡复习标记过的单词
3. 根据掌握程度给出 1-5 星评分

## API Key 设置教程

NotOnlyTranslator 需要调用第三方 AI 翻译服务，因此需要您提供 API 密钥。我们支持以下服务提供商：

### 选项一：OpenAI（推荐）

**优点**：速度快，价格适中，翻译质量优秀

**步骤**：

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册或登录您的 OpenAI 账号
3. 点击左侧菜单 "API keys"
4. 点击 "Create new secret key"
5. 给密钥起个名字（如 "NotOnlyTranslator"）
6. 复制生成的密钥（格式：`sk-...`）
7. 粘贴到 NotOnlyTranslator 设置中

**费用说明**：
- 新用户有 $5 免费额度
- 翻译单词成本约 $0.002/1000 词
- 正常使用每月费用通常 <$1

---

### 选项二：Anthropic Claude

**优点**：翻译质量极高，尤其适合学术内容

**步骤**：

1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 注册或登录您的 Anthropic 账号
3. 在控制台中获取 API 密钥
4. 复制密钥（格式：`sk-ant-...`）
5. 粘贴到 NotOnlyTranslator 设置中

**费用说明**：
- 新用户有 $5 免费额度
- 价格略高于 OpenAI
- 翻译质量更佳

---

### 选项三：自定义 API（高级）

如果您有其他 OpenAI 兼容的 API 端点（如 Azure OpenAI、本地部署等），可以使用自定义选项。

**配置步骤**：

1. 在设置中选择"自定义"
2. 填写 API 端点 URL
3. 输入 API 密钥
4. 选择模型名称
5. 保存设置

---

### API Key 安全说明

🔒 **您的 API 密钥完全安全：**

- 密钥仅存储在您的浏览器本地
- 不会上传到我们的服务器
- 仅用于调用您选择的翻译服务
- 您可以随时删除或更改密钥

## 常见问题 FAQ

### 基础问题

**Q: 为什么需要 API 密钥？**

A: 我们需要调用 AI 翻译服务（如 OpenAI GPT-4o-mini）来分析单词难度和提供翻译。这些服务需要付费，因此我们要求用户自行提供 API 密钥，这样可以：
- 降低扩展本身的成本，保持免费
- 让用户完全控制自己的数据
- 提供更灵活的翻译选项

**Q: API 使用费用高吗？**

A: 不高。正常使用情况下，每月费用通常在 $0.5-$2 之间。OpenAI 和 Anthropic 都提供免费额度，足够试用很长时间。

**Q: 没有 API 密钥能用吗？**

A: 目前不能。翻译功能依赖 AI 服务，必须配置 API 密钥才能使用。

**Q: 支持哪些浏览器？**

A: 目前支持 Chrome、Edge 和其他基于 Chromium 的浏览器。Firefox 版本正在计划中。

---

### 功能问题

**Q: 如何更改我的英语水平？**

A: 点击扩展图标 → "打开设置" → "一般"标签 → 拖动"英语水平"滑块 → 保存。

**Q: 如何标记已掌握的单词？**

A: 鼠标悬停在单词上显示翻译弹窗，点击弹窗右上角的 ✓ 按钮即可标记为已掌握。已掌握的单词将不再高亮显示。

**Q: 误标记了单词怎么办？**

A: 在设置页面的"词汇"标签中，找到该单词，点击"移至学习中"即可。

**Q: 如何导出我的学习数据？**

A: 在设置页面的"数据"标签，点击"导出数据"按钮，可以将所有学习数据下载为 JSON 文件。

**Q: 如何在多台设备同步数据？**

A: 确保您在 Chrome 中登录了 Google 账号并开启同步功能，您的设置和学习数据会自动同步。

---

### 技术问题

**Q: 某些网站上的单词没有被高亮？**

A: 可能原因：
1. 该网站使用了特殊的文本渲染方式（如 Canvas、SVG）
2. 内容是在页面加载后动态加载的（如无限滚动），尝试刷新页面
3. 该内容被识别为代码或非文本内容

**Q: 翻译加载很慢？**

A: 尝试以下方法：
1. 检查您的网络连接
2. 清除翻译缓存（设置 → 数据 → 清除缓存）
3. 考虑更换更快的 API 提供商
4. 大型页面首次加载可能需要更长时间

**Q: 页面显示异常或样式冲突？**

A: 某些网站可能与扩展的高亮样式冲突。您可以：
1. 尝试切换到不同的主题（设置 → 外观）
2. 在该网站上临时禁用扩展
3. 向我们反馈问题

**Q: 如何完全删除扩展和所有数据？**

A:
1. 右键点击扩展图标 → "从 Chrome 中移除"
2. 或者访问 `chrome://extensions/` 找到 NotOnlyTranslator 并删除
3. 这将删除所有本地存储的数据

---

### 隐私问题

**Q: 扩展会收集我的浏览数据吗？**

A: 不会。所有数据都存储在您的本地浏览器中，我们不会收集任何个人信息或浏览历史。

**Q: 网页内容会被发送到服务器吗？**

A: 不会发送完整内容。只有您选择翻译的**单个单词**会发送到您配置的翻译 API，且不含任何上下文信息。

**Q: API 提供商会看到我的数据吗？**

A: 仅发送需要翻译的单个单词，不含网页 URL、用户信息或其他上下文。请参考各 API 提供商的隐私政策了解详情。

## 功能详解

### 智能分级系统

扩展使用贝叶斯推断算法根据您的标记行为动态调整单词难度评估：

- **标记已知**：降低该单词的难度评分
- **标记未知**：提高该单词的难度评分
- **自动调整**：系统会根据您的整体表现微调判定阈值

### 复习算法

采用间隔重复（Spaced Repetition）原理：

- **1星**：几乎忘记，很快会再次复习
- **2星**：有些困难，1天后复习
- **3星**：基本掌握，3天后复习
- **4星**：掌握良好，7天后复习
- **5星**：完全掌握，14天后复习

### 缓存机制

- 翻译结果自动缓存，提高响应速度
- 缓存最多保存 1000 条记录
- 使用 LRU（最近最少使用）策略管理缓存

## 故障排除

### 扩展不工作

**症状**：访问英文网站没有单词被高亮

**检查清单**：

1. ✅ 确认扩展已启用（`chrome://extensions/`）
2. ✅ 确认已配置有效的 API 密钥
3. ✅ 检查英语水平设置是否合理（尝试调整为"初级"测试）
4. ✅ 刷新页面重新加载
5. ✅ 检查浏览器控制台是否有错误（F12 → Console）

**仍不工作？** 请提交 Issue 包含：
- 浏览器版本
- 扩展版本
- 问题网站 URL
- 控制台错误截图

### API 错误

**症状**：显示"翻译失败"或"API 错误"

**解决方案**：

1. 检查 API 密钥是否正确
2. 检查 API 账户是否有足够余额
3. 检查网络连接
4. 尝试更换 API 提供商
5. 检查 API 密钥是否有调用限制

### 性能问题

**症状**：页面卡顿或加载缓慢

**优化建议**：

1. 在设置中调整"每页最大高亮单词数"
2. 定期清理翻译缓存
3. 避免在超大型网页上使用
4. 关闭其他占用资源的扩展
