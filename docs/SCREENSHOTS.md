# Chrome Web Store 截图规格

## 必需截图

Chrome Web Store 需要以下截图来展示扩展功能：

### 1. 主界面截图 (1280x800 或 640x400)

**截图1: 扩展 Popup 界面**
- 尺寸: 640x400px (推荐) 或 1280x800px
- 内容: 显示扩展图标点击后的弹窗界面
- 展示: 用户等级、今日统计、快速操作按钮

**截图2: 网页高亮效果**
- 尺寸: 1280x800px (推荐)
- 内容: 英文网页中部分单词被高亮的效果
- 示例页面: 技术文档或新闻文章
- 展示: 不同颜色表示不同难度的单词

**截图3: 翻译提示框**
- 尺寸: 1280x800px
- 内容: 鼠标悬停在高亮单词上显示翻译提示
- 展示: 单词翻译、音标、标记按钮

### 2. 设置页面截图

**截图4: Options 设置页**
- 尺寸: 1280x800px
- 内容: 完整的设置页面
- 展示: API 配置、水平选择、主题设置等所有选项

**截图5: 统计仪表盘**
- 尺寸: 1280x800px
- 内容: 学习统计页面
- 展示: 词汇掌握度图表、学习趋势、成就

### 3. 特色功能截图

**截图6: 主题切换**
- 尺寸: 1280x800px
- 内容: 暗色模式下的界面效果
- 展示: 暗色主题与亮色主题对比

## 截图设计建议

### 视觉风格
- 使用现代简洁的设计风格
- 突出主要功能，避免过多文字
- 使用高质量的示例内容（真实的英文文章）

### 内容选择
- 选择专业的英文技术文档作为示例
- 展示多种高亮颜色（代表不同难度等级）
- 包含工具提示和交互元素的展示

### 品牌一致性
- 保持与扩展图标一致的颜色方案
- 使用清晰的字体和足够的对比度
- 确保截图在不同尺寸下都清晰可读

## 截图拍摄计划

由于扩展需要在浏览器中实际运行才能截图，建议按以下步骤进行：

1. `npm run build` 构建扩展
2. 在 Chrome 中加载 `dist` 目录
3. 访问英文技术文档网站（如 MDN、React docs）
4. 配置不同的用户等级和主题
5. 使用系统截图工具或 Chrome DevTools 截图
6. 使用图像编辑工具调整至指定尺寸

## 截图质量标准

- 分辨率: 至少 1280x800px（大尺寸截图）
- 格式: PNG（推荐）或 JPEG
- 文件大小: 单张不超过 5MB
- 清晰度: 文字清晰可读，无模糊
