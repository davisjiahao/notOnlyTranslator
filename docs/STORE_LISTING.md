# Chrome Web Store 上架资料

## 扩展名称

NotOnlyTranslator - 智能分级翻译助手

## 简短描述 (最多 132 字符)

智能识别超出你词汇量的单词，只翻译该翻译的内容，让英语学习更自然高效。

## 详细描述

### 简介

NotOnlyTranslator 是一款专为英语学习者设计的智能翻译扩展。它不同于传统的全文翻译工具——它能够识别你的英语水平，只翻译那些超出你当前词汇量的单词，让你在阅读英文内容时保持流畅性的同时，自然地扩展词汇量。

### 核心功能

**🎯 智能分级翻译**
- 根据你的词汇掌握程度，只翻译超出当前水平的单词
- 支持初级、中级、高级和自定义多个难度等级
- 基于贝叶斯算法动态调整词汇掌握度评估

**📚 个性化词汇学习**
- 标记单词为"已掌握"或"未掌握"
- 智能记忆你的词汇学习进度
- 通过阅读积累词汇，而非死记硬背

**🔧 多种翻译引擎支持**
- 支持 OpenAI GPT-4o-mini
- 支持 Anthropic Claude
- 支持自定义 OpenAI 兼容 API
- 更多提供商持续集成中

**📊 学习数据分析**
- 可视化学习进度统计
- 词汇掌握度趋势分析
- 个人学习效率洞察

**⚡ 高性能缓存系统**
- 智能缓存机制减少 API 调用
- LRU 缓存策略，最多缓存 1000 条翻译
- 平均响应时间 < 100ms

**🎨 优雅的界面设计**
- 简洁的单词高亮显示
- 悬浮提示快速查看翻译
- 支持亮色/暗色主题

### 适用场景

- 📖 阅读英文技术文档
- 📰 浏览英文新闻网站
- 📝 学习英文教程
- 💼 处理英文工作邮件
- 🎓 准备英语考试

### 如何使用

1. 安装扩展后，点击图标打开设置
2. 选择你的英语水平或进行词汇量测试
3. （可选）配置 API 密钥以使用翻译服务
4. 开始浏览英文网页，超出你词汇量的单词会自动高亮
5. 点击高亮单词查看翻译，并可标记为已掌握
