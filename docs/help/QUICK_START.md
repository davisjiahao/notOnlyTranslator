# NotOnlyTranslator 快速入门指南

欢迎使用 NotOnlyTranslator！本指南将帮助您在 5 分钟内开始使用这款智能翻译助手。

## 📦 安装

### 从 Chrome Web Store 安装

1. 访问 Chrome Web Store 中的 [NotOnlyTranslator 页面](https://chrome.google.com/webstore/detail/notonlytranslator)
2. 点击"添加至 Chrome"按钮
3. 在弹出的确认对话框中点击"添加扩展程序"

### 手动安装（开发者）

1. 下载最新版本的扩展包
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择解压后的扩展文件夹

## 🔑 第一步：配置 API 密钥

NotOnlyTranslator 需要调用翻译服务 API 来提供智能翻译功能。

### 获取 OpenAI API 密钥

1. 访问 [OpenAI 平台](https://platform.openai.com/)
2. 注册或登录您的账号
3. 进入"API Keys"页面
4. 点击"Create new secret key"
5. 复制生成的密钥

### 获取 Anthropic API 密钥

1. 访问 [Anthropic 控制台](https://console.anthropic.com/)
2. 注册或登录您的账号
3. 进入"API Keys"页面
4. 创建新的 API 密钥
5. 复制生成的密钥

### 配置扩展

1. 点击浏览器工具栏上的 NotOnlyTranslator 图标
2. 点击"选项"进入设置页面
3. 在"翻译提供商"中选择 OpenAI 或 Anthropic
4. 粘贴您的 API 密钥到对应输入框
5. 点击"保存设置"

> 💡 **提示**: API 密钥仅存储在您的本地浏览器中，我们不会也无法访问您的密钥。

## 📊 第二步：设置您的英语水平

1. 在选项页面中，找到"英语水平"设置
2. 拖动滑块选择适合您的水平：
   - **初级**: CET-4 及以下，词汇量约 4,000
   - **中级**: CET-6 / 雅思 5.5-6.5，词汇量约 6,000
   - **高级**: 雅思 7+ / 托福 100+，词汇量约 8,000+
3. 点击"保存设置"

> 💡 **建议**: 初始选择略低于您实际水平的选项，随着使用逐步调整。

## 🌐 第三步：开始使用

### 浏览英文网页

1. 访问任意英文网站（如 BBC News、Medium、Wikipedia）
2. 页面加载完成后，扩展会自动扫描页面内容
3. 超出您水平的单词会被高亮显示

### 查看翻译

1. 将鼠标悬停在高亮显示的单词上
2. 等待片刻，翻译提示框会自动弹出
3. 查看中文翻译、音标和例句

### 标记单词

1. 在翻译提示框中，点击：
   - **✓ 已掌握**: 如果这个词您已经学会
   - **✗ 学习中**: 如果您想记住这个词以便复习
2. 标记会自动保存到您的词汇本中

## 📚 第四步：学习管理

### 查看词汇本

1. 点击扩展图标，选择"选项"
2. 切换到"词汇"标签页
3. 查看"已掌握"和"学习中"的单词列表

### 使用闪卡复习

1. 在选项页面中，点击"复习"标签
2. 系统会根据艾宾浩斯遗忘曲线推送需要复习的单词
3. 点击卡片翻转查看翻译
4. 根据掌握程度给出 1-5 星评分

### 查看学习统计

1. 点击"统计"标签
2. 查看：
   - 词汇量增长曲线
   - 每日学习进度
   - 掌握程度分布
   - 翻译准确率

## ⚙️ 高级设置

### 自定义翻译提示词

1. 在选项页面中，切换到"高级"标签
2. 修改翻译提示词模板
3. 可自定义：
   - 翻译风格
   - 显示内容（音标、例句等）
   - 输出格式

### 主题切换

1. 在选项页面右上角，点击主题切换按钮
2. 选择"浅色模式"或"深色模式"

### 排除特定网站

1. 在"高级"设置中找到"网站排除"
2. 添加不需要启用翻译的网站域名
3. 每行一个域名，例如：`example.com`

## 🎯 使用技巧

### 高效学习

1. **每天标记 10-20 个新词**: 不要贪多，保持可持续性
2. **坚持每日复习**: 系统会在最佳时间推送复习
3. **阅读感兴趣的内容**: 选择自己感兴趣的英文材料，学习效果会更好
4. **标记已掌握**: 如果某词您很熟悉，及时标记，减少重复翻译

### 性能优化

1. **定期清理缓存**: 在选项页面的"存储管理"中清理过期缓存
2. **合理设置英语水平**: 随着词汇量提升，及时调整水平设置
3. **使用翻译缓存**: 重复访问的页面翻译结果会被缓存

## ❓ 常见问题

### 为什么有些单词没有被翻译？

- 可能原因：该单词在您的当前水平范围内，或已被标记为已掌握
- 解决方案：调低英语水平设置，或检查词汇本中的标记

### 翻译加载很慢？

- 检查您的网络连接
- 查看 API 密钥是否有效
- 等待缓存命中（重复单词会直接显示缓存结果）

### 如何导出我的词汇数据？

- 在选项页面的"存储管理"中，点击"导出数据"
- 数据将以 JSON 格式保存到本地
