# NotOnlyTranslator 常见问题 (FAQ)

## 🔑 API 与密钥相关

### Q: 为什么需要 API 密钥？

**A:** NotOnlyTranslator 使用 AI 大模型（如 GPT-4o-mini、Claude）来提供智能翻译服务。这些服务需要 API 密钥来进行身份验证。API 密钥仅存储在您的本地浏览器中，我们不会也无法访问您的密钥。

### Q: 如何获取 OpenAI API 密钥？

**A:**
1. 访问 [OpenAI 平台](https://platform.openai.com/)
2. 注册或登录账号
3. 进入 "API Keys" 页面
4. 点击 "Create new secret key"
5. 复制生成的密钥并粘贴到扩展设置中

### Q: 如何获取 Anthropic API 密钥？

**A:**
1. 访问 [Anthropic 控制台](https://console.anthropic.com/)
2. 注册或登录账号
3. 进入 "API Keys" 页面
4. 点击 "Create Key"
5. 复制密钥并粘贴到扩展设置中

### Q: API 使用会产生费用吗？

**A:** 是的，调用翻译 API 会产生费用，但成本很低：
- OpenAI GPT-4o-mini: 约 ¥0.001-0.002/次翻译
- Anthropic Claude Haiku: 约 ¥0.001-0.003/次翻译
- 新用户通常有免费额度（OpenAI $5, Anthropic $5）

### Q: 我的 API 密钥安全吗？

**A:** 完全安全。API 密钥：
- 仅存储在您的本地浏览器 Chrome Storage 中
- 使用 Chrome 的安全存储机制
- 不会上传至我们的任何服务器
- 我们不会也无法访问您的密钥

### Q: 提示 "API 密钥无效" 怎么办？

**A:**
1. 检查密钥是否复制完整（不要有多余空格）
2. 确认密钥对应的服务商与设置一致
3. 检查 API 账户是否还有余额
4. 如果是新密钥，可能需要等待几分钟生效

## 📊 英语水平设置

### Q: 如何选择合适的英语水平？

**A:** 参考以下对照表：

| 水平 | 适合人群 | 词汇量 |
|------|---------|--------|
| 初级 | CET-4 及以下 | ~4,000 |
| 中级 | CET-6 / 雅思 5.5-6.5 | ~6,000 |
| 高级 | 雅思 7+ / 托福 100+ | ~8,000+ |

**建议**：初始选择略低于实际水平，随着使用逐步提升。

### Q: 为什么有些单词没有被翻译？

**A:** 可能的原因：
1. 该单词在您的当前水平范围内
2. 该单词已被您标记为"已掌握"
3. 该单词是常见的基础词汇（如 the, is, and）

**解决方法**：
- 调低英语水平设置
- 检查词汇本中的标记状态

### Q: 可以自定义词汇水平吗？

**A:** 目前支持通过"词汇管理"功能标记单词的掌握状态。未来版本将支持更细粒度的自定义词汇列表。

## 🌐 翻译功能

### Q: 为什么翻译有时候不准确？

**A:** AI 翻译的准确性取决于：
1. 上下文信息
2. 单词的多种含义
3. 翻译模型的能力

**建议**：
- 结合上下文理解
- 标记翻译不准确的单词，我们会持续优化模型

### Q: 支持哪些网站？

**A:** NotOnlyTranslator 支持几乎所有英文网站，包括：
- 新闻网站（BBC, CNN, Reuters 等）
- 技术文档（GitHub, Stack Overflow, MDN 等）
- 博客平台（Medium, Substack 等）
- 社交媒体（Twitter/X, Reddit 等）

**注意**：某些网站可能有安全限制，导致扩展无法正常工作。

### Q: 翻译加载很慢怎么办？

**A:**
1. **检查网络连接**：确保网络稳定
2. **查看 API 状态**：确认 API 密钥有效且有余额
3. **等待缓存**：重复单词会使用缓存，速度更快
4. **优化设置**：减少同时翻译的单词数量

### Q: 可以翻译整个句子或段落吗？

**A:** NotOnlyTranslator 的设计理念是"只翻译该翻译的"，专注于单词级别的翻译。如需整句翻译，建议使用其他工具如 DeepL 或 Google Translate。

## 📚 词汇学习

### Q: 标记的单词保存在哪里？

**A:** 所有数据都保存在本地：
- **Chrome Sync Storage**：词汇列表和设置（跨设备同步）
- **Chrome Local Storage**：翻译缓存（仅本地）

### Q: 如何备份我的词汇数据？

**A:**
1. 打开扩展选项页面
2. 进入"存储管理"
3. 点击"导出数据"
4. 选择保存位置，数据将以 JSON 格式导出

### Q: 如何导入词汇数据？

**A:**
1. 打开扩展选项页面
2. 进入"存储管理"
3. 点击"导入数据"
4. 选择之前导出的 JSON 文件

### Q: 复习是如何推送的？

**A:** 系统基于 **艾宾浩斯遗忘曲线** 算法：
- 新学习的单词：20分钟后第一次复习
- 之后按 1天 → 2天 → 4天 → 7天 → 15天 的间隔推送

### Q: 可以删除已标记的单词吗？

**A:** 可以：
1. 进入选项页面的"词汇"标签
2. 找到要删除的单词
3. 点击右侧的删除按钮

## 🎨 界面与体验

### Q: 支持深色模式吗？

**A:** 支持！点击选项页面右上角的主题切换按钮即可在浅色/深色模式间切换。

### Q: 可以自定义高亮颜色吗？

**A:** 当前版本不支持自定义颜色，未来版本计划添加此功能。

### Q: 如何在特定网站禁用扩展？

**A:**
1. 打开选项页面
2. 进入"高级"设置
3. 找到"网站排除"
4. 添加需要禁用的域名（每行一个）

### Q: 扩展会影响网页性能吗？

**A:** 经过优化，影响极小：
- 使用 Intersection Observer 实现懒加载
- 翻译结果缓存避免重复请求
- requestAnimationFrame 优化渲染性能

## 🔒 隐私与安全

### Q: 扩展会收集我的浏览数据吗？

**A:** **绝对不会**。我们：
- 不收集任何浏览历史
- 不追踪用户行为
- 不上传网页内容到我们的服务器
- 仅发送需要翻译的**单个单词**到翻译 API

### Q: 我的数据会被用于什么目的？

**A:** 您的数据仅用于：
- 存储您的学习进度
- 优化个性化翻译体验
- 同步到您的其他设备（如开启 Chrome 同步）

### Q: 如何完全删除我的数据？

**A:**
1. 在扩展选项页面中清除所有数据
2. 或直接在 Chrome 中卸载扩展
3. 所有本地数据将被完全删除

## 🐛 故障排除

### Q: 扩展突然不工作了怎么办？

**A:** 尝试以下步骤：
1. 刷新网页
2. 检查扩展是否被禁用（chrome://extensions/）
3. 重新设置 API 密钥
4. 清除浏览器缓存后重启 Chrome

### Q: 显示 "权限不足" 错误？

**A:**
1. 访问 chrome://extensions/
2. 找到 NotOnlyTranslator
3. 点击"详细信息"
4. 确保所有必要权限已开启

### Q: 与其他翻译扩展冲突？

**A:** 建议一次只启用一个翻译扩展。如需使用 NotOnlyTranslator：
1. 禁用其他翻译扩展
2. 刷新页面

**A:**
1. 访问 GitHub Issues: https://github.com/hungrywu/NotOnlyTranslator/issues
2. 点击 "New Issue"
3. 选择 "Bug Report" 模板
4. 详细描述问题和复现步骤

## 💡 使用建议

### Q: 每天学习多少个单词合适？

**A:** 建议每天标记 **10-20 个新词**，保持可持续性比一次性大量学习更重要。

### Q: 如何提高学习效率？

**A:**
1. 阅读感兴趣的内容
2. 坚持每日复习
3. 及时标记已掌握的单词
4. 在不同场景下多次遇到同一单词

### Q: 可以离线使用吗？

**A:** 翻译功能需要联网调用 API，但：
- 词汇本数据可离线查看
- 复习功能可离线使用
- 已缓存的翻译可离线查看
