# Chrome Web Store 截图准备指南

## 截图规格要求

| 规格 | 尺寸 | 格式 | 数量 |
|------|------|------|------|
| **屏幕截图** | 1280x800 或 640x400 | PNG/JPEG | 至少 1 张，最多 5 张 |
| **小图标** | 128x128 像素 | PNG | 1 张（商店列表图标） |
| **推广图片** | 440x280 像素 | PNG/JPEG | 可选，用于商店展示 |
| **横幅图片** | 1400x560 像素 | PNG/JPEG | 可选，用于精选展示 |

## 推荐截图场景（按优先级排序）

### 1. 主界面展示 - 翻译高亮效果 ⭐⭐⭐（必须）
- **场景**: 英文文章页面，显示高亮生词
- **内容**:
  - 选择一篇中等长度的英文文章（如维基百科、新闻）
  - 展示多个高亮单词（不同难度颜色）
  - 显示行内翻译效果
- **浏览器**: Chrome，窗口大小 1280x800
- **操作步骤**:
  1. 打开扩展，确保翻译已启用
  2. 访问英文文章页面
  3. 等待翻译完成，显示高亮
  4. 截取整个浏览器窗口

### 2. Tooltip 翻译详情 ⭐⭐⭐（必须）
- **场景**: 鼠标悬停在高亮单词上显示 Tooltip
- **内容**:
  - 清晰的 Tooltip 展示
  - 显示单词、音标、翻译、例句
  - 难度等级指示
- **操作步骤**:
  1. 在已翻译的页面上
  2. 鼠标悬停在一个高亮单词上
  3. 等待 Tooltip 完全显示
  4. 截取包含 Tooltip 的区域

### 3. 闪卡复习界面 ⭐⭐（推荐）
- **场景**: Options 页面的闪卡复习功能
- **内容**:
  - 显示正面单词
  - 或显示背面翻译和例句
  - 评分按钮可见
- **操作步骤**:
  1. 打开扩展选项页
  2. 进入"闪卡复习"标签
  3. 确保有复习单词
  4. 截取复习界面

### 4. 学习统计仪表盘 ⭐⭐（推荐）
- **场景**: Options 页面的学习统计
- **内容**:
  - 词汇量趋势图
  - 学习热图
  - 掌握度统计
- **操作步骤**:
  1. 打开扩展选项页
  2. 进入"学习统计"标签
  3. 截取统计界面

### 5. API 设置界面 ⭐（可选）
- **场景**: Options 页面的 API 配置
- **内容**:
  - 显示多提供商支持
  - API Key 输入区域
  - 模型选择
- **操作步骤**:
  1. 打开扩展选项页
  2. 进入"API 设置"标签
  3. 截取设置界面

## 截图准备检查清单

### 准备工作
- [ ] 确保扩展已安装并启用
- [ ] 准备测试用的英文文章页面
  - 推荐: https://en.wikipedia.org/wiki/Artificial_intelligence
  - 备选: https://www.bbc.com/news
- [ ] 配置好 API Key（确保翻译功能正常）
- [ ] 准备一些已标记的单词（用于展示学习统计）

### 截图环境
- [ ] 使用 Chrome 浏览器
- [ ] 浏览器窗口调整为 1280x800
- [ ] 隐藏书签栏（Ctrl+Shift+B）
- [ ] 隐藏不必要的扩展图标
- [ ] 使用浅色主题（默认）

### 截图后处理
- [ ] 检查截图清晰度
- [ ] 确保无敏感信息（如个人 API Key）
- [ ] 压缩图片（保持 < 1MB）
- [ ] 按顺序命名: screenshot-1.png, screenshot-2.png 等

## 快速截图脚本

```bash
# 创建截图目录
mkdir -p docs/store-screenshots

# 截图规格检查（需要安装 ImageMagick）
for img in docs/store-screenshots/*.png; do
  echo "检查: $img"
  identify "$img"
done
```

## 截图文件命名规范

```
docs/store-screenshots/
├── screenshot-1-main.png      # 主界面 - 翻译高亮
├── screenshot-2-tooltip.png   # Tooltip 详情
├── screenshot-3-flashcard.png # 闪卡复习
├── screenshot-4-stats.png     # 学习统计
├── screenshot-5-settings.png  # API 设置
└── icon-128x128.png           # 商店图标
```

## 提交前最终检查

- [ ] 所有截图尺寸正确（1280x800 或 640x400）
- [ ] 图片格式为 PNG 或 JPEG
- [ ] 图片清晰，内容可读
- [ ] 无个人敏感信息
- [ ] 展示了扩展的核心功能
- [ ] 至少包含 1 张主界面截图

---

**注意**: Chrome Web Store 可能会自动裁剪或调整图片大小。建议上传前在 Chrome Web Store 开发者后台预览效果。
