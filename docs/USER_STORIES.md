# NotOnlyTranslator 用户故事

## 核心用户画像

### 主要用户：中级英语学习者
- **年龄**: 18-35岁
- **英语水平**: 词汇量 3,000 - 8,000
- **使用场景**:
  - 阅读英文技术文档
  - 浏览英文新闻网站
  - 学习在线课程
- **痛点**:
  - 遇到生词频繁查词典打断阅读
  - 不知道哪些单词值得学习
  - 缺乏学习进度跟踪
- **期望**:
  - 自动识别超出水平的单词
  - 保持阅读流畅性
  - 个性化学习体验

---

## 用户故事详情

### 必须实现 (Must Have)

#### US-001: 智能翻译
**作为**英语学习者，
**我希望**只看到我水平之外的单词翻译，
**以便**保持阅读流畅性。

**验收标准**:
- [ ] 系统根据用户选择的水平（初级/中级/高级）过滤单词
- [ ] 只翻译超出用户水平的单词
- [ ] 翻译以非侵入式方式显示（如颜色高亮）
- [ ] 用户可以悬停查看翻译

---

#### US-002: 查看翻译
**作为**用户，
**我希望**点击翻译后的单词查看释义，
**以便**学习新词汇。

**验收标准**:
- [ ] 点击单词显示工具提示
- [ ] 工具提示包含：中文翻译、词性、简单释义
- [ ] 工具提示样式美观，不遮挡内容
- [ ] 点击外部或按ESC关闭工具提示

---

#### US-003: 标记单词
**作为**用户，
**我希望**标记已学会的单词，
**以便**个性化学习体验。

**验收标准**:
- [ ] 在工具提示中提供"标记为已知"按钮
- [ ] 在工具提示中提供"标记为生词"按钮
- [�] 已标记单词在页面上有视觉区分
- [ ] 标记数据持久化到Chrome Storage

---

#### US-004: 选择水平
**作为**用户，
**我希望**选择我的英语水平（初级/中级/高级），
**以便**获得合适的翻译。

**验收标准**:
- [ ] Popup界面提供水平选择器
- [ ] 选项：初级（词汇量<3000）、中级（3000-8000）、高级（>8000）
- [ ] 选择后立即生效，刷新当前页面
- [ ] 设置持久化到Chrome Storage
