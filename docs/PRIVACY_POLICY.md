# 隐私政策

**生效日期**: 2026年3月21日

**NotOnlyTranslator**（以下简称"本扩展"或"我们"）是一款 Chrome 浏览器扩展，旨在帮助用户根据自身的英语水平智能翻译英文内容。我们高度重视用户的隐私保护，本隐私政策将说明我们如何收集、使用和保护您的信息。

## 1. 我们收集的信息

### 1.1 用户提供的信息
- **API 密钥**: 如果您选择使用 OpenAI、Anthropic 或其他翻译服务提供商，您需要输入相应的 API 密钥。这些密钥仅存储在您的本地浏览器中，使用 Chrome Storage API 进行加密存储，我们不会也无法访问这些信息。
- **用户设置**: 包括您的英语水平设置、主题偏好、翻译偏好等配置信息。

### 1.2 自动收集的信息
- **词汇学习数据**: 扩展会记录您标记为"已掌握"或"未掌握"的单词，用于优化翻译策略。这些数据仅存储在您的本地设备上。
- **翻译缓存**: 为提高性能，翻译结果会被临时缓存（最多1000条），存储在本地浏览器存储中。

## 2. 我们如何收集信息

所有数据收集均通过 Chrome 扩展的本地存储机制完成：
- **Chrome Sync Storage**: 用于跨设备同步用户设置和配置文件（需用户登录 Chrome 账号）
- **Chrome Local Storage**: 用于存储词汇列表和翻译缓存（仅本地存储）

**重要声明**: 我们不会将您的任何数据传输到我们的服务器。

## 3. 信息使用方式

- **API 密钥**: 仅用于调用您选择的第三方翻译服务（OpenAI、Anthropic 等）
- **词汇数据**: 用于计算您的词汇掌握程度，提供个性化的翻译体验
- **设置偏好**: 用于记住您的使用偏好，提供更好的用户体验

## 4. 第三方服务

本扩展使用以下第三方服务：

| 服务提供商 | 用途 | 隐私政策 |
|-----------|------|---------|
| OpenAI | 翻译服务（可选） | [OpenAI 隐私政策](https://openai.com/privacy) |
| Anthropic | 翻译服务（可选） | [Anthropic 隐私政策](https://www.anthropic.com/privacy) |

**注意**: 使用这些服务时，您的文本内容会发送到相应的服务提供商进行处理。请查阅他们的隐私政策了解详情。

## 5. 数据存储与安全

- 所有数据存储在您的本地浏览器中
- API 密钥使用 Chrome 的安全存储机制
- 我们不维护任何服务器或数据库来存储用户数据
- 词汇和设置数据可选择通过 Chrome 同步功能在您的设备间同步

## 6. 您的权利

您可以随时：
- 在扩展设置中清除所有本地数据
- 删除已保存的 API 密钥
- 导出或删除您的词汇数据
- 卸载扩展以删除所有相关数据

## 7. 儿童隐私

本扩展不面向13岁以下儿童。我们不会有意收集儿童的个人信息。

## 8. 隐私政策更新

我们可能会不时更新本隐私政策。任何变更将在扩展更新时通知您，并在本页面上发布新的隐私政策。

本扩展需要以下 Chrome 权限：

| 权限 | 用途 |
|------|------|
| `storage` | 保存用户设置和词汇数据 |
| `activeTab` | 在当前页面高亮需要翻译的单词 |
| `contextMenus` | 提供右键菜单快捷操作 |
| `scripting` | 在网页中注入高亮样式 |
| `host_permissions` | 在访问的网页上使用内容脚本 |
