# NotOnlyTranslator 产品路线图

## 产品愿景

NotOnlyTranslator 是一个智能 Chrome 浏览器扩展，根据用户的英语水平智能翻译内容。它只翻译超出用户当前水平的单词，帮助英语学习者自然阅读，同时扩展词汇量。

---

## 用户故事

### 核心用户画像
- **目标用户**: 中级英语学习者（词汇量 3000-8000）
- **使用场景**: 阅读英文网页、新闻、技术文档
- **痛点**: 遇到生词时频繁查词典打断阅读流畅性
- **期望**: 自动识别并翻译超出自己水平的单词

### 用户故事列表

#### 必须实现 (Must Have)
1. **作为**英语学习者，**我希望**只看到我水平之外的单词翻译，**以便**保持阅读流畅性
2. **作为**用户，**我希望**点击翻译后的单词查看释义，**以便**学习新词汇
3. **作为**用户，**我希望**标记已学会的单词，**以便**个性化学习体验
4. **作为**用户，**我希望**选择我的英语水平（初级/中级/高级），**以便**获得合适的翻译

#### 应该实现 (Should Have)
5. **作为**用户，**我希望**选择不同的LLM提供商（OpenAI/Anthropic/自定义），**以便**灵活性和成本控制
6. **作为**用户，**我希望**设置自己的API密钥，**以便**隐私保护
7. **作为**用户，**我希望**查看我的学习进度统计，**以便**跟踪学习效果
8. **作为**用户，**我希望**翻译结果有视觉区分（颜色/下划线），**以便**快速识别

#### 可以实现 (Could Have)
9. **作为**用户，**我希望**在特定网站上禁用扩展，**以便**灵活性
10. **作为**用户，**我希望**导出我的学习数据，**以便**备份或迁移
11. **作为**用户，**我希望**听到单词发音，**以便**学习正确发音
12. **作为**用户，**我希望**看到例句，**以便**理解单词用法

#### 暂不实现 (Won't Have)
13. **作为**用户，**我希望**离线翻译（此版本不涉及）
14. **作为**用户，**我希望**社交学习功能如好友竞赛（此版本不涉及）

## 成功指标

### 技术指标
- 翻译响应时间 < 1秒（缓存命中）
- 翻译响应时间 < 3秒（缓存未命中，调用LLM）
- 内容脚本加载时间 < 100ms
- Chrome Storage使用 < 5MB

### 用户体验指标
- 用户留存率（7天/30天）
- 日均翻译单词数
- 用户词汇量增长跟踪
- 用户满意度评分

### 业务指标
- 活跃用户数量
- 用户增长率
- API成本每用户

## 技术风险与缓解策略

| 风险 | 影响 | 缓解策略 |
|-----|-----|---------|
| LLM API延迟高 | 用户体验差 | 实现积极缓存策略 |
| API密钥泄露 | 安全风险 | 使用Chrome Storage加密，本地存储 |
| 大页面DOM操作性能差 | 浏览器卡顿 | 虚拟化/分批处理 |
| LLM成本超预算 | 运营成本高 | 缓存、批处理、用户配额 |
| Chrome API变更 | 扩展失效 | 跟随Manifest V3最佳实践 |
