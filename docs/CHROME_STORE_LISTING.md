# NotOnlyTranslator - Chrome Web Store 上架素材

## 产品信息

### 名称
NotOnlyTranslator - 智能分级翻译助手

### 简短描述（最多 132 字符）
智能识别您的英语水平，只翻译超出您词汇量的单词，让英语学习更自然高效。

### 详细描述

**NotOnlyTranslator 是一款专为英语学习者设计的智能翻译插件。**

不同于传统翻译工具将整个页面翻译成中文，NotOnlyTranslator 会智能评估您的英语水平，只翻译那些超出您当前词汇量的单词。这样，您可以在阅读英文内容时保持英语思维，同时逐步扩展词汇量。

**核心功能：**

🎯 **智能分级翻译**
- 基于您的词汇量水平，精准识别需要翻译的单词
- 支持初级、中级、高级多个难度等级
- 自动适应您的学习进度

📚 **个性化词汇管理**
- 标记已掌握和正在学习的单词
- 智能复习系统，基于艾宾浩斯遗忘曲线
- 闪卡式单词复习功能

📊 **学习进度追踪**
- 可视化学习数据统计
- 词汇量增长曲线
- 每日学习打卡

🔧 **灵活的翻译设置**
- 支持 OpenAI、Anthropic Claude 等多个 AI 翻译提供商
- 可自定义 API 密钥
- 支持自定义翻译提示词

🌐 **全网页支持**
- 自动扫描页面内容
- 悬停查看翻译
- 支持主流英文网站

**适合人群：**
- 正在提升英语阅读能力的学习者
- 需要阅读英文资料但不想完全依赖翻译的学生和职场人士
- 希望扩大词汇量的英语爱好者

**隐私保护：**
- 所有数据存储在本地浏览器
- 不会收集您的阅读内容
- 仅将需要翻译的单词发送至翻译 API

**使用说明：**
1. 安装插件后，点击图标设置您的 API 密钥（支持 OpenAI 或 Anthropic）
2. 在选项页面选择您的英语水平
3. 访问英文网站，插件会自动高亮并翻译超出您水平的单词
4. 点击单词查看翻译，标记已掌握或待学习

**反馈与支持：**
如有问题或建议，请通过 GitHub Issues 联系我们。
