### 详细描述

## NotOnlyTranslator - 你的智能英语阅读助手

读英文文章时，是不是经常这样？
- 遇到生词要切出去查词典，读完后上下文忘了
- 用全文翻译，确实不用查词了，但你根本没在读英文
- 试了各种翻译工具，要么太简单，要么太复杂

### 核心功能

**智能分级翻译**
- 根据你的英语水平自动判断哪些词需要翻译
- 只翻译你不会的，已掌握的词汇保持原文
- 在阅读中自然区分已知/未知词汇

**上下文保留**
- 原文和翻译同时显示，随时对照
- 点击即可查看翻译，不打断阅读节奏
- 悬浮窗设计，不遮挡页面内容

**词汇积累**
- 自动记录你查询过的生词
- 标记已掌握词汇，优化翻译策略
- 支持词汇导出，配合 Anki 等工具复习

**隐私保护**
- 使用你自己的 API Key，数据完全私有
- 支持本地 LLM，无需联网
- 无追踪，无广告

### 适合谁用

- **考研党**：真题阅读、外刊精读
- **四六级考生**：提升阅读速度和词汇量
- **留学预备**：学术论文、申请资料阅读
- **职场人士**：技术文档、商务邮件
- **英语爱好者**：原版书籍、新闻网站

### 如何使用

1. 安装扩展后点击图标打开设置
2. 配置你的 OpenAI / Anthropic / DeepL / Google Translate API Key
3. 设置你的英语水平（CET-4/CET-6/考研/雅思等）
4. 访问任意英文网页，自动生效
