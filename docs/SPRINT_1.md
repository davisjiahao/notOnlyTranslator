# Sprint 1 计划

## 概述

- **Sprint 名称**: Sprint 1 - 核心功能实现
- **时间盒**: 2周 (2026-03-14 至 2026-03-28)
- **目标**: 完成 Must Have User Stories (US-001 至 US-004)，实现可工作的翻译演示

---

## 用户故事与任务分解

### US-001: 智能翻译 (优先级: Critical)

**作为**英语学习者，**我希望**只看到我水平之外的单词翻译，**以便**保持阅读流畅性。

**验收标准**:
- [ ] 系统根据用户选择的水平过滤单词
- [ ] 只翻译超出用户水平的单词
- [ ] 翻译以非侵入式方式显示（颜色高亮）
- [ ] 用户可以悬停查看翻译

**技术任务**:

| ID | 任务 | 负责人 | 估计工时 | 依赖 |
|----|------|--------|----------|------|
| US-001-T1 | 实现页面段落扫描器 (PageScanner) | Founding Engineer | 4h | - |
| US-001-T2 | 实现词汇过滤服务 (基于用户水平) | Founding Engineer | 6h | US-001-T1 |
| US-001-T3 | 实现高亮渲染器 (Highlighter) | Founding Engineer | 4h | US-001-T2 |
| US-001-T4 | 集成 Background Script 翻译 API | Founding Engineer | 4h | US-001-T3 |

---

### US-002: 查看翻译 (优先级: Critical)

**作为**用户，**我希望**点击翻译后的单词查看释义，**以便**学习新词汇。

**验收标准**:
- [ ] 点击单词显示工具提示
- [ ] 工具提示包含：中文翻译、词性、简单释义
- [ ] 工具提示样式美观，不遮挡内容
- [ ] 点击外部或按ESC关闭工具提示

**技术任务**:

| ID | 任务 | 负责人 | 估计工时 | 依赖 |
|----|------|--------|----------|------|
| US-002-T1 | 实现 Tooltip 组件 (React + Tailwind) | Founding Engineer | 6h | US-001-T3 |
| US-002-T2 | 实现点击事件处理和定位逻辑 | Founding Engineer | 4h | US-002-T1 |
| US-002-T3 | 集成翻译详情获取 (Background API) | Founding Engineer | 2h | US-002-T2 |
| US-002-T4 | 添加 ESC 键关闭和点击外部关闭 | Founding Engineer | 2h | US-002-T2 |

---

### US-003: 标记单词 (优先级: High)

**作为**用户，**我希望**标记已学会的单词，**以便**个性化学习体验。

**验收标准**:
- [ ] 在工具提示中提供"标记为已知"按钮
- [ ] 在工具提示中提供"标记为生词"按钮
- [ ] 已标记单词在页面上有视觉区分
- [ ] 标记数据持久化到 Chrome Storage

**技术任务**:

| ID | 任务 | 负责人 | 估计工时 | 依赖 |
|----|------|--------|----------|------|
| US-003-T1 | 实现标记服务 (MarkerService) | Founding Engineer | 4h | US-002-T1 |
| US-003-T2 | 在 Tooltip 中添加标记按钮 | Founding Engineer | 2h | US-003-T1, US-002-T1 |
| US-003-T3 | 实现 Chrome Storage 存储层 | Founding Engineer | 4h | US-003-T1 |
| US-003-T4 | 实现已标记单词的视觉区分 | Founding Engineer | 2h | US-003-T3 |

---

### US-004: 选择水平 (优先级: High)

**作为**用户，**我希望**选择我的英语水平，**以便**获得合适的翻译。

**验收标准**:
- [ ] Popup 界面提供水平选择器
- [ ] 选项：初级、中级、高级
- [ ] 选择后立即生效，刷新当前页面
- [ ] 设置持久化到 Chrome Storage

**技术任务**:

| ID | 任务 | 负责人 | 估计工时 | 依赖 |
|----|------|--------|----------|------|
| US-004-T1 | 扩展 Popup UI 添加水平选择器 | Founding Engineer | 4h | - |
| US-004-T2 | 实现用户配置存储 (Zustand + Chrome Storage) | Founding Engineer | 4h | US-004-T1 |
| US-004-T3 | 实现水平变更事件通知 Content Script | Founding Engineer | 2h | US-004-T2 |
| US-004-T4 | 页面刷新后应用新水平设置 | Founding Engineer | 2h | US-004-T3 |

| 词汇识别准确性 | 中 | 中 | 使用成熟的 NLP 库 |

## 完成的定义 (Definition of Done)

每个任务必须满足以下条件才算完成：

- [ ] 代码实现完成并通过 ESLint
- [ ] TypeScript 类型检查通过
- [ ] 单元测试覆盖率 > 80%
- [ ] E2E 测试通过（如适用）
- [ ] 代码审查通过
- [ ] 文档更新（如需要）

---

## Sprint 评审标准

Sprint 1 成功完成的标准：

1. ✅ 用户可以在英文页面上看到超出其水平单词的高亮
2. ✅ 用户可以点击高亮单词查看翻译详情
3. ✅ 用户可以标记单词为已知/生词
4. ✅ 用户可以在 Popup 中选择/修改英语水平
5. ✅ 所有设置和标记数据持久化保存
