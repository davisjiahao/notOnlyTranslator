# Chrome Web Store 截图准备清单

## 截图规格要求

Chrome Web Store 要求：
- **尺寸**: 1280x800 (推荐) 或 1440x900
- **格式**: PNG 或 JPEG
- **数量**: 至少 1 张，推荐 3-5 张
- **大小**: 每张不超过 5MB

## 需要准备的截图

### 1. 主界面展示 (main-interface.png)
**场景**: 在英文网页上使用插件的效果
**内容**:
- 打开一个英文文章页面（如 Medium、BBC News）
- 显示高亮的单词（超出用户水平的单词）
- 显示鼠标悬停时的翻译提示框
- 展示"已掌握"/"学习中"标记按钮

**步骤**:
1. 安装扩展并配置 API 密钥
2. 设置英语水平为"中级"
3. 访问 https://medium.com 或 https://www.bbc.com/news
4. 等待页面扫描完成，显示高亮单词
5. 鼠标悬停在一个高亮单词上显示翻译
6. 截图（确保分辨率为 1280x800）

---

### 2. 设置页面 (settings-page.png)
**场景**: 扩展选项页面的设置界面
**内容**:
- API 配置面板
- 英语水平选择
- 主题设置

**步骤**:
1. 点击扩展图标，选择"选项"
2. 进入设置页面
3. 展示"翻译提供商"设置（OpenAI/Anthropic/自定义）
4. 展示"英语水平"滑块
5. 截图

---

### 3. 统计面板 (statistics-dashboard.png)
**场景**: 学习统计页面
**内容**:
- 词汇量增长曲线
- 今日学习统计
- 掌握程度分布

**步骤**:
1. 在选项页面点击"统计"标签
2. 确保有一些学习数据（可以先标记一些单词）
3. 展示图表和数据
4. 截图

---

### 4. 闪卡复习 (flashcard-review.png)
**场景**: 闪卡式单词复习界面
**内容**:
- 单词卡片正面（英文单词）
- 点击翻转后的背面（中文翻译）
- 评分按钮（1-5星）

**步骤**:
1. 在选项页面点击"复习"标签
2. 显示单词卡片
3. 截图正面
4. 点击卡片显示背面翻译
5. 截图背面（可选：第二张图）

---

### 5. 词汇管理 (vocabulary-management.png)
**场景**: 词汇列表页面
**内容**:
- "已掌握单词"标签页
- "学习中单词"标签页
- 搜索和筛选功能

**步骤**:
1. 在选项页面点击"词汇"标签
2. 标记一些单词为已掌握和学习中
3. 展示词汇列表
4. 截图

## 截图技巧

1. **浏览器设置**
   - 使用 Chrome 浏览器
   - 窗口大小设置为 1280x800
   - 隐藏书签栏（Ctrl+Shift+B）
   - 使用隐身模式避免插件干扰

2. **页面选择**
   - 选择内容丰富的英文网站
   - 避免有敏感内容的页面
   - 确保页面加载完全后再截图

3. **视觉效果**
   - 确保高亮颜色清晰可见
   - 翻译提示框不要被遮挡
   - 保持界面整洁
