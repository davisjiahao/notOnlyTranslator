## P0 优化项：翻译引擎指示器

### 用户问题
用户不知道翻译使用了哪个引擎，无法判断翻译质量和来源。

### 解决方案
在翻译结果中显示翻译引擎和耗时。

### 交互规格

```
┌─────────────────────────────────────────────────────────────┐
│ 翻译结果卡片                                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  "Hello world"                                              │
│  ↓                                                          │
│  "你好世界"                                                  │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ [DeepL] · 0.3s  │ [LLM] · 1.2s  │ [Youdao] · 0.2s │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 设计规格

| 元素 | 规格 | 说明 |
|------|------|------|
| 引擎标签 | 字号 10px, 颜色 #6B7280 | 次要信息，不干扰阅读 |
| 耗时 | 字号 10px, 颜色 #9CA3AF | 灰色显示，精确到 0.1s |
| 分隔符 | "·" 符号 | 中点分隔 |

### 状态显示逻辑

| 翻译类型 | 显示内容 |
|----------|----------|
| 传统翻译 (DeepL/Youdao) | 引擎名 + 耗时 |
| LLM 翻译 | "LLM" + 耗时 + 可能的"高质量"标签 |
| 混合翻译 | 两个引擎都显示 |
| 缓存命中 | "缓存" + 无耗时 |

### 代码位置建议
- 文件: `src/content/tooltip.tsx`
- 组件: TranslationResult

## P0 优化项：加载状态反馈

### 用户问题
翻译过程中无反馈，用户不知道是否在工作，产生焦虑。

### 解决方案
添加骨架屏或加载动画。

### 交互规格

#### 状态 1: 初始加载 (0-500ms)
```
┌─────────────────────────────────────────┐
│ ████████████                            │  ← 灰色脉冲动画
│ ████████████████████                    │
└─────────────────────────────────────────┘
```

#### 状态 2: 翻译中 (500ms+)
```
┌─────────────────────────────────────────┐
│ 正在翻译...                             │
│ [████████░░░░░░░░░░] 45%                │  ← 进度条
└─────────────────────────────────────────┘
```

#### 状态 3: 复杂文本 (LLM 翻译)
```
┌─────────────────────────────────────────┐
│ 🧠 正在进行深度分析...                   │
│ 这是一段复杂的文本，我们会仔细翻译       │  ← 安抚文案
└─────────────────────────────────────────┘
```

### 动画规格
- 脉冲周期: 1.5s
- 颜色: #E5E7EB → #F3F4F6
- 缓动: ease-in-out

### 代码位置建议
- 文件: `src/content/tooltip.tsx`
- 新增: LoadingState 组件

## P0 优化项：错误提示优化

### 用户问题
错误信息过于技术化，用户不知道如何解决。

### 现状
```
❌ Error: API key not configured (code: AUTH_001)
```

### 优化后
```
┌─────────────────────────────────────────────────────────────┐
│ ⚠️ 翻译暂时不可用                                           │
│                                                             │
│ 您的 API 密钥可能需要更新                                    │
│                                                             │
│ [前往设置]  [查看帮助]                                       │
└─────────────────────────────────────────────────────────────┘
```

### 错误类型映射

| 错误类型 | 用户友好文案 | 行动按钮 |
|----------|--------------|----------|
| API Key 无效 | "翻译服务需要验证" | [前往设置] |
| 网络错误 | "网络连接不稳定" | [重试] |
| 配额用尽 | "翻译额度已用完" | [查看方案] |
| 超时 | "翻译时间较长" | [等待] / [取消] |
| 未知错误 | "翻译遇到问题" | [重试] / [反馈] |

### 代码位置建议
- 文件: `src/background/translation.ts`
- 新增: UserFriendlyErrors 映射表

## P1 优化项：预翻译策略

### 用户问题
点击翻译后有等待感，体验不够即时。

### 解决方案
后台预翻译可视区域内的文本。

### 实现策略

```javascript
// 预翻译优先级
const pretranslationPriority = {
  // 高优先级：可视区域
  visible: {
    threshold: 1.0,  // 完全可见
    action: 'immediate'  // 立即预翻译
  },
  // 中优先级：即将进入视口
  nearVisible: {
    threshold: 0.5,  // 部分可见
    action: 'debounce 500ms'  // 延迟预翻译
  },
  // 低优先级：视口外
  offscreen: {
    threshold: 0,
    action: 'skip'  // 不预翻译
  }
};
```

### 缓存策略
- 预翻译结果缓存 30 分钟
- 用户实际点击时检查缓存有效性
- 缓存命中时立即显示

## P1 优化项：学习进度可视化

### 用户问题
用户不清楚自己的学习进展，缺乏成就感。

### 解决方案
添加学习进度面板。

### 设计规格

```
┌─────────────────────────────────────────────────────────────┐
│ 📚 我的学习                                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  今日已学: 12 个单词                                        │
│  ████████████░░░░░░░░ 60% 目标完成                          │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 本周学习曲线                                         │   │
│  │    ▁▂▃▅▇█▆▄▃                                       │   │
│  │  周一 周二 周三 周四 周五 周六 周日                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  已掌握: 342 词  |  学习中: 28 词  |  待复习: 15 词         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 实现要点
1. 使用 chrome.storage.local 存储学习数据
2. 图表使用 SVG 绘制
3. 每日学习目标默认 20 词，可自定义

## 验收标准

### 翻译引擎指示器
- [ ] 翻译结果显示引擎名称
- [ ] 显示翻译耗时
- [ ] 缓存命中显示"缓存"标签

### 加载状态反馈
- [ ] 点击后立即显示加载状态
- [ ] 骨架屏/动画流畅

### 错误提示优化
- [ ] 错误信息用户友好
- [ ] 提供明确的解决行动
- [ ] 包含帮助链接
