### Onboarding 研究核心洞察

**问题识别**: NotOnlyTranslator 的 API Key 配置门槛高于所有竞品

| 竞品 | API Key 要求 | 门槛 |
|------|-------------|------|
| Google Translate | 无需 | 极低 |
| Readlang | 无需 | 低 |
| Language Reactor | 可选账户 | 中 |
| 沉浸式翻译 | 内置额度 | 中 |
| **NotOnlyTranslator** | **必需** | **高** ⚠️ |

**优化建议优先级**:

🔴 **高优先级 - Quick Wins**
1. 简化 API Key 配置流程（一键配置、验证提示）
2. 添加互动首次使用教程
3. 优化 Popup 首次打开体验

🟡 **中优先级 - 短期改进**
4. 提供试用额度（100次免费翻译）
5. 用户英语水平评估
6. 渐进式功能解锁

🟢 **低优先级 - 长期战略**
7. 智能推荐系统
8. 社区引导
