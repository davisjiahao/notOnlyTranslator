# 用户 Onboarding 访谈大纲

## 访谈信息

| 项目 | 内容 |
|------|------|
| **访谈目标** | 了解新用户首次使用 NotOnlyTranslator 的体验，识别 onboarding 过程中的痛点 |
| **目标用户** | 近7天内安装扩展的新用户 |
| **访谈时长** | 30-45 分钟 |
| **记录方式** | 录音（经用户同意）+ 笔记 |

---

## 开场白（5分钟）

> 您好，感谢您参与这次访谈。我是 NotOnlyTranslator 团队的用户研究员，正在进行一项关于新用户体验的研究。
>
> 这次访谈大约需要 30-45 分钟，目的是了解您第一次使用我们产品的体验。您的反馈将帮助我们改进产品，让新用户更容易上手。
>
> **保密声明**：您的所有回答都将严格保密，仅用于产品改进目的。我们不会将您的个人信息与任何第三方分享。
>
> **录音许可**：您是否同意我们录音以便后续分析？（可以随时停止录音）
>
> 如果您准备好了，我们可以开始了。有任何问题随时问我。

---

## 第一部分：背景了解（5分钟）

### 1.1 用户背景
1. 您能否简单介绍一下您的背景？（职业、学习或工作领域）
2. 您的英语水平大概是什么程度？
3. 您平时经常需要阅读英文内容吗？主要是哪种类型的内容？

### 1.2 使用场景
4. 在安装 NotOnlyTranslator 之前，您是如何处理英文阅读中的生词的？
5. 是什么让您决定尝试我们的扩展？（从哪里了解到？）

---

## 第二部分：安装与首次体验（10分钟）

### 2.1 安装体验
6. 安装扩展的过程顺利吗？有没有遇到任何问题？
7. 安装完成后，您做的第一件事是什么？

### 2.2 首次打开
8. 您第一次点击扩展图标时，看到了什么？
9. 那个时候您知道接下来该做什么吗？

---

## 第三部分：API Key 配置（10分钟）

### 3.1 配置体验
10. 您是如何了解到需要配置 API Key 的？
11. 配置 API Key 的过程是怎样的？能描述一下步骤吗？
12. 在这个过程中，您遇到了什么困难吗？

### 3.2 困难与障碍
13. （如遇到困难）具体是什么让您感到困惑？
14. 您是如何解决这些问题的？（搜索、询问、尝试等）
15. 如果没有配置成功，是什么让您最终放弃？

### 3.3 改进建议
16. 您认为 API Key 配置过程可以如何改进？
17. 您希望有什么样的引导或帮助？

---

## 第四部分：首次翻译体验（10分钟）

### 4.1 首次使用
18. 您第一次使用翻译功能是在什么网页上？
19. 第一次使用时的体验如何？符合您的预期吗？

### 4.2 功能探索
20. 您发现了哪些功能？（翻译、单词标记、统计等）
21. 有没有哪些功能是您期望有但没找到的？

### 4.3 困惑点
22. 使用过程中，有哪些时刻让您感到困惑或不确定？
23. 您希望在使用初期得到什么样的引导？

---

## 第五部分：整体评价与建议（5分钟）

### 5.1 满意度评价
24. 总的来说，您对第一次使用 NotOnlyTranslator 的体验满意吗？（1-10分）
25. 您认为这个产品的最大优点是什么？
26. 最需要改进的地方是什么？

### 5.2 推荐意愿
27. 您会向朋友推荐这个产品吗？为什么？

### 5.3 改进建议
28. 如果让您给新用户写一条使用建议，您会写什么？
29. 您还有什么其他反馈或建议想要分享的吗？

## 追问问题库

### 当用户提到困难时
- 能具体描述一下当时的情况吗？
- 您当时有什么感受？
- 您尝试了哪些解决方法？
- 如果有一个更好的解决方案，您希望是什么样的？

### 当用户提到期望时
- 您期望的功能是什么样的？
- 为什么这个功能对您重要？
- 您在其他产品上看到类似的功能吗？

### 当用户不确定时
- 您当时的第一反应是什么？
- 您希望在那个时刻看到什么样的提示？
- 如果有向导功能，您希望它如何引导您？
