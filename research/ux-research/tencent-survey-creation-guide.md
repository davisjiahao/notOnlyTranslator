# 腾讯问卷创建指南

**创建截止**: 2026-03-22
**负责人**: UX Researcher
**决策来源**: CEO (2026-03-21)

---

## 🎯 问卷目标

招募 NotOnlyTranslator 新用户参与产品体验访谈，了解首次使用体验。

**目标人数**: 8-10 人
**访谈时长**: 30-45 分钟
**激励**: 30元京东卡 或 1个月高级会员

---

## 📝 创建步骤

### Step 1: 登录腾讯问卷

1. 访问 https://wj.qq.com
2. 使用微信或 QQ 登录
3. 点击「创建问卷」

### Step 2: 选择问卷类型

选择「空白问卷」或「从模板创建」

### Step 3: 设置问卷标题

```
标题: NotOnlyTranslator 用户访谈招募
描述:
感谢您使用 NotOnlyTranslator！我们正在改进产品体验，诚邀您参与简短访谈。
访谈时长：30-45 分钟
参与奖励：30元京东卡 或 1个月高级会员
```

---

## 📋 问卷字段设计

### 必填字段（5个）

#### 问题 1: 微信号/手机号
```
类型: 单行文本
标题: 您的微信号或手机号（用于联系安排访谈）
必填: ✅
验证: 手机号格式（可选）
```

#### 问题 2: 英语学习时长
```
类型: 单选题
标题: 您学习英语多长时间了？
必填: ✅
选项:
- 1年以下
- 1-3年
- 3-5年
- 5-10年
- 10年以上
```

#### 问题 3: 是否已安装扩展
```
类型: 单选题
标题: 您是否已经安装了 NotOnlyTranslator 扩展？
必填: ✅
选项:
- 是，已安装并在使用
- 是，已安装但还没用过
- 否，还没安装（跳转结束页）
```

**逻辑跳转**: 选择「否」的用户跳转到结束页，显示：
```
感谢您的关注！本次访谈主要面向已安装用户。
欢迎您先安装体验我们的产品，后续会再次招募。
```

#### 问题 4: 使用频率
```
类型: 单选题
标题: 您使用 NotOnlyTranslator 的频率是？
必填: ✅
选项:
- 每天都用
- 每周几次
- 偶尔使用
- 只用过1-2次
- 刚安装还没用过
```

#### 问题 5: 方便访谈时间
```
类型: 多选题
标题: 您方便接受访谈的时间段是？（可多选）
必填: ✅
选项:
- 工作日上午（9:00-12:00）
- 工作日下午（14:00-18:00）
- 工作日晚间（19:00-22:00）
- 周末上午
- 周末下午
- 周末晚间
```

#### 问题 6: 主要使用场景
```
类型: 多选题
标题: 您主要在哪些场景使用 NotOnlyTranslator？（可多选）
必填: ❌
选项:
- 阅读技术文档/教程
- 阅读学术论文

- 工作/邮件沟通
- 其他：_____
```

#### 问题 7: 对产品的期望
```
类型: 多行文本
标题: 您希望 NotOnlyTranslator 有哪些改进或新功能？
必填: ❌
字数限制: 200字以内
```

## 📊 数据收集与筛选

### 筛选标准

| 条件 | 要求 | 筛选逻辑 |
|------|------|----------|
| 安装状态 | 已安装 | 问题3 = "是" |
| 使用频率 | 至少偶尔 | 问题4 ≠ "还没用过" |
| 联系方式 | 有效 | 问题1 有内容 |

### 优先级排序

1. **高优先级**: 每天使用 / 每周几次 + 英语学习 3-10年
2. **中优先级**: 偶尔使用 / 只用过1-2次
3. **低优先级**: 刚安装还没用过（可能不符合条件）

### 用户分组目标

| 职业 | 目标人数 |
|------|----------|
| 开发者/技术人员 | 2-3 人 |
| 学生 | 2-3 人 |
| 职场人士 | 2-3 人 |
| 其他 | 1-2 人 |
