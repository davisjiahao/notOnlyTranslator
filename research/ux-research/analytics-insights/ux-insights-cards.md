# UX Research 洞察卡片

## 洞察 #1: 翻译准确性是用户最关注的核心价值

**来源**: 用户访谈 (n=15) + 问卷调研 (n=120)
**日期**: 2026-03-15
**相关画像**: 进阶英语学习者、学术研究者

### 洞察描述
用户在选择翻译插件时，翻译准确性是他们最看重的因素。78% 的受访者将"翻译准确性"列为首要考虑因素，远超"界面美观"(45%) 和"功能丰富"(52%)。用户期望翻译不仅能准确传达字面意思，还要能根据上下文提供恰当的释义。

### 证据
- 用户访谈中，12/15 位用户明确提到翻译准确性是他们选择插件的首要标准
- 问卷调研中，78% 的用户将"翻译准确性"列为最重要的功能特性
- 用户原话："我宁愿界面简单一点，但翻译一定要准" (P3)
- 用户原话："有时候翻译不准反而误导我，还不如自己去查词典" (P7)

### 影响
- **产品影响**: 翻译质量是核心竞争力，需要持续投入优化
- **业务影响**: 翻译准确性直接影响用户满意度和留存率
- **技术影响**: 需要持续优化 LLM 提示词，考虑引入术语库和上下文理解

### 建议行动
- [ ] 建立翻译质量评估体系，定期测试和优化
- [ ] 投资 LLM 提示词工程，提升翻译准确性和上下文理解
- [ ] 考虑引入用户反馈机制，收集翻译错误报告
- [ ] 研究术语库和专业领域翻译优化方案

### 相关洞察
- 洞察 #2: 用户对词汇学习功能的粘性很高
- 洞察 #5: 高级用户需要更多自定义选项

## 洞察 #2: 词汇学习功能具有高用户粘性

**来源**: 使用数据分析 (30天, n=500) + 用户访谈 (n=8)
**日期**: 2026-03-10
**相关画像**: 进阶英语学习者、职场提升者

### 洞察描述
使用闪卡复习功能的用户相比不使用该功能的用户，留存率高出 35%，日活跃时长增加 42%。词汇学习功能将翻译工具转化为学习工具，创造了持续使用的价值。用户表示，看到每天标记的单词被系统性地用于复习，增强了学习的成就感和持续性。

### 证据
- 使用数据分析：使用闪卡功能的用户 7 日留存率为 68%，不使用者为 33%
- 用户平均每周使用闪卡功能 4.2 次，平均每次 12 分钟
- 用户原话："每天花 10 分钟复习，比背单词书有意思多了" (P5)
- 用户原话："能看到自己的词汇量在增长，很有动力" (P2)

### 影响
- **产品影响**: 词汇学习是产品的差异化功能，应持续强化
- **业务影响**: 高粘性功能带来更好的用户留存和口碑传播
- **技术影响**: 需要优化学习算法，个性化复习节奏

### 建议行动
- [ ] 扩展学习功能，增加拼写测试、例句填空等模式
- [ ] 优化间隔重复算法，提高学习效率
- [ ] 增加学习进度可视化，增强成就感
- [ ] 考虑添加学习提醒功能，培养使用习惯

### 相关洞察
- 洞察 #1: 翻译准确性是用户最关注的核心价值
- 洞察 #4: 用户希望获得学习进度反馈

## 洞察 #3: API Key 配置是新用户的主要门槛

**来源**: 用户访谈 (n=12) + 可用性测试 (n=5)
**日期**: 2026-03-12
**相关画像**: 进阶英语学习者、职场提升者

### 洞察描述
API Key 的配置过程是用户 onboarding 过程中最大的摩擦点。60% 的受访用户在首次配置时遇到困难，平均需要 15-20 分钟完成配置。用户普遍反馈获取和配置 API Key 的过程技术门槛较高，需要更清晰的分步指引和视觉辅助。

### 证据
- 可用性测试：5/5 名用户在配置 API Key 时遇到困难
- 用户平均在配置页面停留 18 分钟
- 用户原话："我不知道 OpenAI API key 是什么，也不知道怎么获取" (P4)
- 用户原话："配了半天没成功，差点就放弃了" (P6)

### 影响
- **产品影响**: 高门槛导致新用户流失，影响转化率
- **业务影响**: 用户获取成本增加，首次体验受损
- **技术影响**: 需要重新设计配置流程

### 建议行动
- [ ] 提供详细的图文 API Key 获取指南
- [ ] 添加交互式配置向导，分步骤引导
- [ ] 考虑提供内置 API 选项（免费试用或付费）
- [ ] 添加配置验证功能，实时检查配置是否正确

## 洞察 #4: 用户需要可视化的学习进度反馈

**来源**: 问卷调研 (n=150) + 用户访谈 (n=10)
**日期**: 2026-03-14
**相关画像**: 进阶英语学习者、职场提升者

### 洞察描述
82% 的用户表示希望看到可视化的学习进度反馈，包括词汇量增长曲线、学习热力图、掌握度分布等。可视化反馈不仅满足用户的好奇心，更重要的是创造了成就感和持续使用的动力。用户特别希望看到长期趋势，了解自己的进步轨迹。

### 证据
- 问卷调研：82% 的用户希望看到可视化进度反馈
- 用户最希望的统计图表类型：词汇量趋势 (78%)、学习热力图 (65%)、词性分布 (45%)
- 用户原话："看到自己的词汇量增长曲线，比任何鼓励的话都管用" (P3)
- 用户原话："希望能看到一年的学习轨迹，知道自己的进步" (P7)

### 影响
- **产品影响**: 可视化功能将显著提升用户体验和留存
- **业务影响**: 增强的成就感带来更好的用户口碑
- **技术影响**: 需要设计数据可视化方案和性能优化

### 建议行动
- [ ] 设计词汇量增长曲线图（支持日/周/月视图）
- [ ] 添加学习热力图，展示每日学习活动
- [ ] 创建词汇掌握度分布图（饼图/雷达图）
- [ ] 设计成就徽章系统，奖励学习里程碑
- [ ] 支持数据导出，让用户拥有自己的学习数据

### 相关洞察

## 洞察 #5: 高级用户需要更多自定义选项

**来源**: 深度访谈 (n=8) + 用户反馈分析
**日期**: 2026-03-16
**相关画像**: 学术研究者、进阶英语学习者（高级）

### 洞察描述
高级用户（英语 CET-6/雅思 7.0+）表现出对深度自定义的强烈需求，包括自定义翻译规则、词汇难度标准、专业术语库等。这些用户通常有明确的学习目标和专业领域，通用的"一刀切"设置无法满足其精细化需求。他们愿意为高级功能付费。

### 证据
- 深度访谈：8/8 名高级用户提到需要更多自定义选项
- 用户反馈中，"自定义"是高频出现的需求词（出现率 45%）
- 用户原话："我学的是医学英语，希望能导入医学术语库" (P2)
- 用户原话："如果能自定义翻译规则，我愿意付费使用" (P5)

### 影响
- **产品影响**: 需要开发高级功能模块，区分免费和付费用户
- **业务影响**: 开辟付费功能/专业版的可能
- **技术影响**: 需要设计灵活的自定义配置系统

### 建议行动
- [ ] 开发高级设置面板，提供更多自定义选项
- [ ] 支持自定义词汇列表（导入/导出）
- [ ] 设计专业术语库系统，支持多领域
- [ ] 考虑开发"专业版"付费计划
- [ ] 创建 Beta 测试项目，邀请高级用户参与
