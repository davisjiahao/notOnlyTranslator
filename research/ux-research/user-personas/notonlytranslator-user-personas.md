# NotOnlyTranslator 用户画像

## 主要用户画像

### 👤 画像 1：进阶英语学习者 "英语进阶者小李"

**基本信息**
- **年龄**: 24-30 岁
- **职业**: 互联网行业从业者（产品经理/程序员/设计师）
- **教育背景**: 本科及以上
- **英语水平**: CET-6 / 雅思 6.0-6.5 / 托福 80-90

**行为特征**
- 每天阅读英文技术文档、博客、新闻
- 使用英语学习类 App（如百词斩、扇贝）
- 在工作中需要阅读或撰写英文邮件
- 偶尔观看英文技术视频或参加英文会议

**目标与动机**
- 提升专业英语词汇量，特别是技术术语
- 保持英语阅读能力，为可能的海外机会做准备
- 更高效地获取英文信息，减少对翻译的依赖
- 通过真实语境学习，而非死记硬背单词

**痛点与挑战**
- 遇到生词需要频繁查词典，打断阅读流畅性
- 无法判断哪些词值得学习，哪些词可以跳过
- 难以坚持传统的单词背诵方式
- 不确定自己的词汇量水平和进步情况

**使用场景**
1. 早上阅读英文技术博客时，用插件辅助理解难词
2. 工作中阅读英文文档，快速理解关键术语
3. 晚上复习当天标记的单词，巩固记忆

**对产品的期望**
- 智能判断词汇难度，只翻译真正需要帮助的词
- 简洁不干扰阅读体验的界面设计
- 方便的单词标记和复习功能
- 能看到自己的词汇量进步

### 👤 画像 2：学术研究者 "科研工作者王博士"

**基本信息**
- **年龄**: 28-40 岁
- **职业**: 博士研究生 / 博士后 / 青年研究员
- **领域**: 理工科（计算机、生物、化学、物理等）
- **英语水平**: CET-6 / 雅思 6.5-7.5

**行为特征**
- 每天阅读大量英文学术论文
- 需要撰写英文学术论文和报告
- 参加国际学术会议，阅读英文会议资料
- 使用文献管理工具（如 Zotero、EndNote）

**目标与动机**
- 高效获取国际前沿研究信息
- 扩展学术英语词汇量，特别是专业术语
- 提升英文学术写作能力
- 为国际合作和交流做准备

**痛点与挑战**
- 学术论文词汇量大，专业术语多
- 阅读效率受限，难以快速把握论文要点
- 难以积累和复习学术词汇
- 需要频繁使用词典，影响阅读连贯性

### 👤 画像 3：职场提升者 "职场精英张女士"

**基本信息**
- **年龄**: 28-35 岁
- **职业**: 外企员工 / 跨国公司职员 / 商务人士
- **行业**: 金融、咨询、市场、贸易等
- **英语水平**: CET-4/6，工作中实际使用英语

**行为特征**
- 工作需要处理英文邮件、报告
- 阅读英文商业新闻、行业报告
- 参加英文会议或电话会议
- 需要准备英文演示文稿

**目标与动机**
- 提升商务英语沟通能力
- 扩大商务词汇量，特别是行业术语
- 增强职场竞争力
- 为职业发展或跳槽做准备

**痛点与挑战**
- 商务英语词汇不够丰富
- 阅读英文商业资料速度较慢
- 难以准确理解某些商务表达
- 缺乏系统性的词汇学习方法

## 用户画像使用指南

### 在产品设计中应用

1. **功能优先级排序**：根据主要画像的需求确定功能开发优先级
2. **界面设计参考**：考虑画像的技术水平和使用场景设计界面
3. **文案撰写**：使用画像熟悉的语言和术语

### 在营销中应用

1. **内容创作**：针对画像的痛点和需求创作内容
2. **渠道选择**：根据画像的信息获取习惯选择营销渠道
3. **信息传递**：突出画像最关心的产品价值

### 定期更新

用户画像应定期更新，建议：
- 每季度审查一次画像的准确性
- 收集新用户数据时补充画像信息
- 根据产品发展阶段调整画像优先级
